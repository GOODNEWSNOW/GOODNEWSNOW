# /今日与恩典同行/

### 第133 05/13 礼拜五

>信就是所望之事的实底，是未见之事的确据。古人在这信上得了美好的证据.…人非有信，就不能得神的喜悦；因为到神面前来的人必须信有神，旦信他赏赐那寻求他的人。——希伯来书11:1-2、6

我们如果在与基督的关系中没有信心，就好像是我家孩子小时候不敢过吊桥一样。桥上的地锚和吊索都很牢固，但我两个年幼的孩子过桥时却战战競兢！他们极为煎熬地在桥上爬行，即使旁边不时有人走过。

这就好像是那些对自己在基督里的安稳没有信心的人。他们在追求圣洁的途中缓缓往前爬行，无力抵挡困难和试探的风暴。

今日就站稳了，因为你与基督联合使你能满有胆量和自信地行在信心中。这皆是因基督为你所行的一切！

**祷告：主耶稣，求你帮助我今日有胆量且自信地凭信而行，因为你与我同在。**
