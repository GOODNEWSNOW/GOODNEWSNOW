# /今日与恩典同行/

### 第101日 04/11 礼拜一

>各人要照所得的恩赐彼此服侍，作神百般恩赐的好管家。若有讲道的，要按着神的圣言讲；若有服侍人的，要按着神所赐的力量服侍，叫神在凡事上因耶稣基督得荣耀。 -彼得前书4:10-11

一则旧车打蜡广告描绘了一位年轻女子准备卖车。她的车破旧不堪，黯淡无光，魅力不再。可当她用“奇迹”蜡使表面重现光泽，重新焕发的色彩让她又爱上了自己的车，于是她就开车走了，一边走一边扔掉“出售”的牌子。

这则广告道出了深刻的道理。我们爱自己所投资的东西。当我们为他人的满足感、安全感和发展而付出时，我们所做的不仅仅是建立他们的自尊——我们把自己的自尊加给了他们。

所以，今天就为别人的价值感增值吧。让他们因你微笑，你也更加喜爱他们！

**祷告：主啊，这周请帮助我用你给我的恩赐去服侍和投资于他人特别是那些在基督身体里的人。**
