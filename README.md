# /今日与恩典同行/

### 第137 05/17 礼拜二

>伯利恒以法他啊，你在犹大诸城中为小，将来必有位从你那里出来，在以色列中为我作掌权的；他的根源从亘古，从太初就有。——弥迦书5:2

当旧约先知弥迦预言我们的弥赛亚在伯利恒诞生的时候，他强调这个小城并不算什么——在犹大人中甚至不值一提。但这个名字却藏着隐含的意义。你知道“伯利恒”在希伯来语中的意思是粮食之家吗？耶稣称自己为“生命的粮”，所以这真是个巧妙的文字游戏。

粮食可以很好地代表耶稣，因为我们人人都需要粮食。在这个比喻中，我们看到神因着他的恩典，差遣他的儿子成为从天上来的粮——使所有人都能得到，为拯救所有人。耶稣的话正宣告了这点：“我就是生命的粮，到我这里来的，必定不饿。

**祷告：主啊，我感谢你不仅供应我每天的粮食，还借着你的儿子耶稣基督给了我生命的粮！**
