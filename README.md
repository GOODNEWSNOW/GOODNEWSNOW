# /今日与恩典同行/

### 第150 05/30 礼拜一

>因有一婴孩为我们而生；有一子赐给我们。政权必担在他的肩头上；他名称为“奇妙策士、全能的神、永在的父、和平的君”。——以赛亚书9:6

有时候我们都会有一些好的参谋我们身边有人能够理解我们的需要。在圣经中，耶稣被称为奇妙策士，但更好的翻译是：“他要成为奇迹谋士”。奇迹谋士能做什么呢？嗯，首先，他给人好的建议。但这种建议只有在谋士深深理解我们时才会出现。幸运的是，我们的“奇迹谋士”耶稣完完全全地了解我们。他创造了我们，他知道我们的心。他知道你的心。

因为神对你的大爱和恩典，耶稣与你认同。在你需要的时刻，他是唯一能真正理解和安慰你的人。

**祷告：耶稣，我何等感激，因你对我的认识远超过我对自己的认识。求你在我需要的时候赐给我智慧、恩典和安慰。**
