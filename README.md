# /今日与恩典同行/

### 第164 06/13 礼拜一

>这事谁肯依从你们呢？上阵的得多少，看守器具的也得多少；应当大家平分。——撒母耳记上30:24

珍妮特 (Janette Alexander） 是19世纪伟大的神学家阿奇博尔德 (Archibald Alexander）的妻子。她对圣经有深刻的见解，她家人因此尤为珍视她。她量然不像她丈夫受过正式的训练，她的儿子（也是知名圣经教师）却常常向她请教，因为他相信她的常识比世上所有的注释书都更宝贵。

在这个美好的互补搭配中，我们得以从神完全的视角看见一人的价值。珍妮特不是神学院教授，她只是一位妻子、母亲和神的孩子。我们的生命彼此交织，这让我们看到，那使我们能服侍神的人就和服侍的人一样重要。而在神的恩典中，两样人他都会使用！

**祷告：主啊，求你帮助我用你所赐我的恩赐服侍你。愿这些恩赐能用来建立基督的身体。**
