# /今日与恩典同行/

### 第177 06/26 主日

>我的弟兄们，你们落在百般试炼中，都要以为大喜乐：因为知道你们的信心经过试验，就生忍耐。但忍耐也当成功，使你们成全、完备，毫无缺欠。——雅各书1:2-4

你曾试过在试炼中赞美神吗？这似乎不合常理。但是赞美神能生出勇气来面对我们的对手。赞美也能使我们的心思集中在神的伟大上，而当我们专注在神而非自己的问题上时，我们的试炼就变小了。我们赞美神的时候试炼并末消失，但试炼中的赞美能给我们勇气去忍耐。

所以，当生命似乎分崩离析之时，以赞美亲近神，到神的恩典和平安之下。在那里，在他慈爱的看顾之下，你会经历到他是你的盼望，你的盾牌，他也能使你重拾喜乐！

**祷告：父啊，你知道我不喜欢生命中的试炼或艰难。可是我要赞美你，因为试炼和艰难使我更像基督。**
