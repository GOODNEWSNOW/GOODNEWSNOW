# /今日与恩典同行/

### 第181 06/30 礼拜四

>你们既作顺命的儿女，就不要效法从前蒙昧无知的时候那放纵私欲的样子。那召你们的既是圣洁，你们在一切所行的事上也要圣洁。因为经上记着说：“你们要圣洁，因为我是圣洁的。”——彼得前书1:14-16

彼得前书告诉我们：“你们要圣洁，因为我是圣洁的。”我们的生命中若没有基督，这些话听上去就会很可怕！你会好奇，神怎样期待我成为圣洁？尽管我们有许多缺点和罪，神赐给我们丰盛的爱，并因着他儿子耶稣基督已做成的工把我们当作圣洁。

你看，神的恩典首先使我们成为圣洁，然后使我们得自由，可以出于对神的爱和感恩而去追求圣洁。你永远都不需要担心你的“好行为”是否足以换取神的接纳，因他接纳你不是基于你的功劳，而是在乎耶稣基督的宝血和公义。

**祷告：父啊，我知道你己经为我披戴基督的公义和圣洁。但是，求你帮助我每一天都靠着你圣灵的大能，成为圣洁，就像你圣洁那样。**
