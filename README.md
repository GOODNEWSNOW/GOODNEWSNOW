# /今日与恩典同行/

### 第142 05/22 主日

>若有人在基督里，他就是新造的人，旧日事已过，都变成新的了。——哥林多后书5:17

在哥林多后书S章17节中，使徒保罗说：“若有人在基督里，他就是新造的人。，但成为新造的人并不意味着你会突然拥有职业运动员的身体或先知的头脑。保罗说的是我们灵性上发生的变化。

例如，在我们与基督联合之前，我们无法停止犯罪，因为犯罪是我们本性的一部分！但借着基督且在基督里，我们的本性被改变。我们披上了基督的公义，我们从罪的权势中得到了释放。因为基督的慈爱看顾，我们现在有能力抵挡罪恶，追求圣洁！

这是神恩典的大能明证，这恩典使你我成为新造的人！

**祷告：天父，求你今天帮助我通过我的思想、言语和行动确认，我是在基督里新造的人。**
