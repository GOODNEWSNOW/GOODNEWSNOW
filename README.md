# /今日与恩典同行/

### 第118日 04/28 礼拜四

>愿颂赞归与我们主耶稣基督的父神！他曾照自己的大怜悯，借耶稣基督从死里复活，重生了我们，叫我们有活泼的盼望，可以得着不能朽坏、不能玷污、不能衰残、为你们存留在天上的基业。——彼得前书1:3-4

前不久，我参加了一个非常豪华的宴会。在穿着燕尾服、打着黑领带的上流社会人士中，我感到格格不入。这时，一个身穿破旧牛仔裤、旧T恤和登山靴的男子向我走来，与我搭起了话。

尽管这人看上去很贫穷，穿着也很随意，但他却散发着自信，后来我才知道为什么。他是一大笔财产的继承人。虽然他父亲当时和他断绝了来往，但他知道，他最终还是会得到他的产业。所以这人并没有因为自己暂时的身份而苦恼。

同样，我们也要自信而行，因为我们是智慧慈爱的天父所赐永恒产业的继承人。

**祷告：父啊，感谢你赐给我天上永恒的产业，因为你在基督里赐下了永恒的生命！**
