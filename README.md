# /今日与恩典同行/

### 第141 05/21 礼拜六

>你们中间谁有儿子求饼，反给他石头呢？求鱼，反给他蛇呢？你们虽然不好，尚且知道拿好东西给儿女，何况你们在天上的父，岂不更把好东西给求他的人吗？——马太福音7:9-11

神说我们要把大小祈求都带到他面前，这是多好的事啊！在段艰难的时期，这个真理击中了我的心。我弟弟在牢房里祈求神帮助我的父母找到他那只跑丢的狗。

我不得不承认，我当时心里想：“不要为这个祷告！神有更大的问题要处理。”我父亲还是去了老红前一周跑掉的停车场。他喊了几次老红的名字都没有反应，心想：“这太荒唐了，我在做什么？”然后，当他转身走回车里时，就看到了摇着尾巴的老红！

神的恩典覆盖了我们生命中的一切，这是多好的提醒。所以，向神发出你的祈求吧。他在意。他在听。

**祷告：天父，当我今天来到你面前祈祷时，求你帮助我认识到你乐意赐好礼物给你儿女。**
