# /今日与恩典同行/

### 第105日 04/15 礼拜五

>有一件事，我曾求耶和华，我仍要寻求：就是一生世住在耶和华的殿中，瞻仰他的荣美，在他的殿里求问。 -诗篇27:4

我永远不会忘记我和妻子的第一次“约会”。当时我和她的家人一起去郊游。我是他们那间小乡村教会的新任单身牧师，受邀参加他们家的野餐，野餐地在伊利诺伊州的艾尔萨 (Elsah），那是个维多利亚式的复古小镇。

那个金发碧眼的漂亮姑娘邀我“散步”，我马上答道：“没问题！”她的美丽叫我欣喜地跟随着她。

同样，耶稣也向我们显出他的美好，表明他对我们的爱和恩典。所以，当耶稣说“跟从我”，救主的美好让人不由自主地就去跟从他。我们乐意与耶稣同行，因为我们明白他是何等可爱。

**祷告：主耶稣，谢谢你美好而荣耀的性情。求你帮助我一整天都欢喜跟随你。**
