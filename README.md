# /今日与恩典同行/

### 第145 05/25 礼拜三

>因有一婴孩为我们而生；有一子赐给我们。政权必担在他的肩头上；他名称为“”奇妙策士、全能的神、永在的父、和平的君”。——以赛亚书9:6

以赛亚书开头几章回顾了以色列历史上一个危急而昏暗的时期。神的子民背弃了他，希望之光似乎完全熄灭了.

这肯定会是个悲伤的故事——如果结局就是如此的话。但事实不是这样！以赛亚书第9章提醒我们，神有一个计划，要让他的子民回到他身边！以赛亚书宣称：“因有一婴孩为我们而生，有一子赐给我们。”他将成为世界的救主！

这就是我们在圣诞节和全年所庆祝的荣耀恩典。正是因为耶稣才有了这个节日，也正是因为耶稣，我们的罪没有成为我们故事的结局。这真是一个值得庆祝的理由。

**祷告：天父，感谢你没有让我的罪成为故事的结局。求你帮助我每天庆祝和平之子的降临。**
