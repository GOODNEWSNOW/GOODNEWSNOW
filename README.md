# /今日与恩典同行/

### 第131 05/11 礼拜三

>亲爱的弟兄啊，我们现在是神的儿女，将来如何，还未显明；但我们知道，主若显现，我们必要像他，因为心得见他的真体。凡向他有这指望的，就洁净自己，像他洁净一样。 ——约翰一书3:2-3

最近，我晨跑的途中会看到一块牌子。上面说，有一块士地将被分割成18个新住宅用地。这个提议似乎很荒设，凡是看到这个牌子的人都认为，要在那块如此没有吸引力的土地上建房真是疯了。我们无法想象其中的潜力，因为我们没有用建筑师的眼光来看那片土地。对建筑师来说，这块土地的价值不在乎它目前的情形而是它末来可以成为什么样子。

同样，神也不会受我们目前的状况所限制。他有一个美好计划，要在我们身上并通过我们成就他的旨意。所以，请记住，你的价值不取決于现在或过去，而在乎将来。

**祷告：天父，我盼望有一天可以面对面看见耶稣基督，变得完全像一样。愿这盼望能激励我每天洁净自己。**
