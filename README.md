# /今日与恩典同行/

### 第166 06/15 礼拜三

>只要以里面存着长久温柔、安静的心为妆饰；这在神面前是极宝贵的。因为古时仰赖神的圣洁妇人正是以此为妆饰，顺服自己的丈夫。——彼得前书3:4-5

神从一开始就赋予了女人极大的尊荣和价值，而这个世界仍不明白这一点。这个世界教导我们，如果一个女人没有做到公司高层，或者没有达到一定的薪资和地位，那她就不如那些达到这些条件的女人有价值。

可是这样来衡量一个女人就背离了神充满恩典的旨意和计划。因为我们在圣经中看到，一个女人的尊荣展现在她为所服侍的人带来的祝福——无论在家、在教会还是在职场。她的荣耀在于她情愿照神所看重的次序行事，她关注的焦点在他人身上，天国就此满有恩典地彰显出来。

**祷告：主啊，愿我每日关注的焦点是服侍他人，并在我的言行上反映你所看重的次序。**
