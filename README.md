# /今日与恩典同行/

### 第108日 04/18 礼拜一

>我的神必照他荣耀的丰富，在基督耶稣里，使你们切所需用的都充足。 -腓立比书4:19

多年来，乔治•穆勒 (George Mueller） 从末向任何人要过钱来资助他的孤儿院。他单单依靠祷告来满足他们的需要，而神一直在供应。但在1842年3月9日，情况急转直下，他们的资源耗尽了。

就在一切就要用尽的时候，来了一封快递信。信封里有一份几天前寄出的数量可观的捐项。虽然原先送错了地方，但这封信的到来，意味着主在几天前就已经开始回应穆勒的祈祷。

神把所有的思想、事件、时间安排交织在一起，使捐项得以在所需的关键时刻到达。我们要以此为鉴。神在我们最需要的时候，为我们的益处而编织万事！

**祷告：天父，感谢你总是丰富地供应我的需要。请帮助我在资源似乎耗尽时依然相信你。**
