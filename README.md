# /今日与恩典同行/

### 第98日 04/08 礼拜五

>各人要照所得的恩赐彼此服侍，作神百般恩赐的好管家。 -彼得前书4:10

镇上新开了一家洪尔玛，那里迅速成为当地的社交中心——人人都想去的地方。有一个叫弗洛伦斯的女人，她几乎把这家店当成了自己第二个家，而她丈夫比尔则把湖泊当成了自己第二个家。

后来她不能再自己开车了，比尔就牺牲自己的湖边时间，带着妻子进城。大家都知道这个男人热爱户外运动，但后来又逐渐看到，这个男人甘愿牺牲自己的娱乐时间来服侍妻子。

当比尔坐在超市中间的草坪椅上，他向自己的社区示范了耶稣牺牲的爱，也向他的妻子展示了恩典和爱。

**祷告：耶稣，就像你为了救赎我而献上生命来服侍我，愿我也能舍己地服侍他人。**
