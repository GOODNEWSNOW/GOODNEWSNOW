# /今日与恩典同行/

### 第152 06/01 礼拜三

>基督耶稣降世，为要拯救罪人。这话是可信的，是十分可佩服的。在罪人中我是个罪魁。然而，我蒙了怜悯，是因耶稣基督要在我这罪魁身上显明他一切的忍耐，给后来信他得永生的人作榜样。——提摩太前书1:15-16

你听过《史上最佳圣诞汇演》 (The Best Christmas Pageant Ever）这部戏剧吗？故事讲的是一个十二岁的女孩，她名叫伊莫金，是个十足的不良少女。伊莫金想尽一切办法说服教会汇演的导演，他们应该绑架真正的婴儿来扮演耶稣的角色。虽然导演向伊莫金解释，绑架婴儿是不对的，但刚愎自用的她根本听不进去。

可演出还得继续下去。于是，伊莫金就全力以赴表演马利亚的角色。在表演的过程中，她被真正的圣诞故事所改变同时被神对她的爱和恩典所折服。

这真是好消息！因为，如果神能救赎像伊莫金这样的问题女孩，那么凭着他的恩典，他也能救赎我们中的任何一个！

**祷告：天父，如果你能救像保罗和伊莫金那样的人，那么我就知道你能救任何人——包括我！谢谢你在基督里的怜悯和恩典。**
