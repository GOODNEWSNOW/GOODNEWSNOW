# /今日与恩典同行/

### 第114日 04/24 主日

>因我们并不是与属血气的争战，乃是与那些执政的、掌权的、管辖这幽暗世界的，以及天空属灵气的恶魔争战。所以，要拿起神所赐的全副军装。 —以弗所书6:12-13a

许多人对使徒保罗在以弗所书第6章中所说魔鬼的黑暗势力作了分类。大多数分类都纯属推测，而保罗想告诉我们，撒但是通过这幽暗世界的罪恶势力在工作——那些势力就在我们眼前。

败坏的领袖、堕落的追求、贫穷和物质主义。那恶者装作光明的天使四处游走，叫我们看不见这些势力的破坏作用，又用自私自利迷惑我们，使我们分心。我们往往会为个人的自由、权力或快乐而接受这些罪恶。

但保罗提醒我们，不要被敌人的权势和无处不在的欺骗影响了。相反，我们要紧紧抓住我们对独一真神的信心！

**祷告：天父，求你帮助我不被撒但的罪恶势力所迷惑或分心。愿你的属灵军装保护我免受敌人的伤害！**
