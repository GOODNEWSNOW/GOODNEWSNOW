# /今日与恩典同行/

### 第136 05/16 礼拜一

>推罗王希兰曾照所罗门所要的，资助他香柏木、松木，和金子；所罗门王就把加利利地的二十座城给了希兰。希兰从推罗出来，察看所罗门给他的城邑，就不喜悅。——列王纪上9:11-12

知名基督徒作家伊丽莎白•艾略特 (Elisabeth Eliott） 讲述了一个年轻女孩的故事，她在大萧条期问失去了父亲——再过四天就是她17岁生日。女孩知道，没有父亲养家，自己的生活将不再一样。而周围的寂静似乎压得她喘不过气来。但她随后听到——有什么东西冲破了压迫，向她说话。

说话的是扫昂，她母亲三天没用过的扫昂。这个简单的工具所说的比她母亲能说出的任何话都要多。它说，生活还要继续。我们已经信靠了主。我们会坚持下去。我们会重新活过来。一把寂寞的扫昂以如此有力又如此温柔的方式，在曾经失去信心的地方重新点燃信心。

**祷告：天父，我有时希望你能以激动人心、显而易见的方式对我说话。可是请帮助我在你轻声低语时听见你的声音。**
