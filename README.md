# /今日与恩典同行/

### 第139 05/19 礼拜四

>我的口终日要述说你的公义和你的救恩，因我不计其数。我要来说主耶和华大能的事；我单要提说你的公义。神啊，自我年幼时，你就教训我；直到如今，我传扬你奇妙的作为。——诗篇71:15-17

前段时间，圣约神学院的高级董事会成员吉姆•奥德斯 (Jim Orders）因患癌住院。我去看望他时，他说自己相信神对他的计划，还说：“我一直以来都想知道天父会如何带我回家。

在最后的日子里，吉姆还有最后一件事想完成。他想写一本关于他家族历史和生意的书，以领赞神在他生命中的恩典。他想通过回忆录表达自己对救主的热心，因为他渴望为自己所爱的人留下一份遗训，让他们也能对救主产生深深的爱和热心。

那么，神的恩典在你的生命里有什么故事呢？你不用再等待分享的时机。也许今天就有人需要听到它。

**祷告：主啊，求你帮助我记住你在我生命中所做的一切美好之事使我能把你的爱和恩典告诉别人。**
