# /今日与恩典同行/

### 第173 06/22 礼拜三

>倚靠耶和华、以耶和华为可靠的，那人有福了！他必像树栽于水旁，在河边扎根，炎热来到，并不惧怕，叶子仍必青翠，在千旱之年毫无挂虑，而且结果不止。——耶利米书17:7-8

圣经应许我们，若忠心顺服神就会得奖赏，这些经文激励了我们很多人。但这些祝福不是我们善行的功劳，而是神信实的凭据。神以仁慈、恩典和怜悯将这些祝福赐给他所爱的人，好叫全世界都看到他杰出的工作。

家庭合睦、个人幸福和经济稳定只是其中几个例子，显明了尊荣神的旨意和标准通常会带来的祝福。更大的祝福是神操练和转变我们的品格，叫我们更像耶稣。这些神的祝福都是从神的恩典和怜悯流溢而出，不是因我们良善，而是因他良善。

**祷告：父啊，愿你在我们生命中的丰盛祝福激励我生出敬虔的品格。**
