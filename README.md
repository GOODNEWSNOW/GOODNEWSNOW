# /今日与恩典同行/

### 第153 06/02 礼拜四

>你们当刚强壮胆，不要害怕，也不要畏惧他们，因为耶和华你的神和你同去。他必不撇下你，也不丢奔你。——申命记31:6

几十年前，有部分美国人质被扣押在伊朗，这种情况一直持续到圣诞期间。迫于来自全球的政治压力，恐怖分子允许人质一起庆贺圣诞。那时，美国大使馆工作人员凯瑟琳-库布(Kathryn Koob） 虽然仍被关押，却充满了盼望，轻轻地哼唱起《马槽歌》。

最后一段歌词对她而言异常深刻与宝贵：“恭敬求主耶穌，靠近我身旁，爱护我。”凯瑟琳•库布所寻得的盼望正是你我也可以得着的盼望。神差他儿子以马内利到我们中间，因着他的恩典神与我们同在！

今日记得要喜乐，并要常常喜乐，因为知道神必不撇下你。

**祷告：主啊，我深知无论我的生命遭遇何种艰难或困苦，我的盼望都在基督里。谢谢你的应许：你必不会撇下我，也不会丢弃我！**
