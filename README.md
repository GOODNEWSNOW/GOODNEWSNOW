# /今日与恩典同行/

### 第175 06/24 礼拜五

>这不是说我已经得着了，已经完全了；我乃是竭力追求，或者可以得着基督耶稣所以得着我的。——腓立比书3:12

我们跟随基督，就必须不断与罪争战，可这并不表示我们在今生能达致完全。我们无法做到完全，因为只有一个人能做到，那就是我们救主耶稣基督。当我们明白神对我们的爱何等的大，就会不禁想要为他而活。我们能这么做是因着圣灵的能力，他充满我们、引导我们，又使我们成为圣洁，更像耶稣。

我们自己的努力永远没法使我们完全，但正因此故，我们仰望为我们信心创始成终的耶稣”。因着他恩典的牺牲，我们才能抵挡罪，才能为他的荣耀而活！

**祷告：主啊，你知道我远不及完全。但求你借着你圣灵的能力，使我能更像耶稣。**
