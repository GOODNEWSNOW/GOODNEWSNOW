# /今日与恩典同行/

### 第146 05/26 礼拜四

>一切苦毒、恼恨、忿怒、嚷闹、毁谤，并一切的恶毒，都当从你们中间除掉；并要以恩慈相待，存怜悯的心，彼此饶恕，正如神在基督里饶恕了你们一样。——以弗所书4:31-32

在废除种族隔离制度后的南非，一名虚弱的黑人妇女站在法庭上，面对着杀害她丈夫和儿子的男子。法官问她：你想要什么？，她的回答让旁听的人震惊不已。

她回答说：“我想要三件事。我想知道我丈夫的尸体在哪里，这样我就可以把他好好地埋葬了。我想让被告作我的儿子，这样他就可以来看我，我就可以向他倾注我剩余的爱。最后，我想让他知道，我饶恕了他，因为耶稣正是为饶恕而死。

一个智慧女人说了多么有力的话！她能够给那个谋杀她心爱之人的男子恩典，因为她已经完全领受了神的恩典。而恩典不仅改变我们，也能改变别人！

**祷告：主啊，因为我已经完全领受了你的恩典，求你帮助我以饶恕和爱待那些伤害我的人。**
