# /今日与恩典同行/

### 第147 05/27 礼拜五

>牧羊的人彼此说：“我们往伯利恒去，看看所成的事，就是主所指示我们的。”他们急忙去了，就寻见马利亚和约瑟，又有那婴孩卧在马槽里。_路加福音2:15-16

我们许多人都是唱着圣诞领歌《哦，伯利恒的小城》(Oh,little town of Bethlehem） 长大的。但我们可能对歌词太过熟悉，以至于不再对其中的信息感到惊叹。

耶稣出生的时候，伯利恒已经成了无名小城从“大卫之城^的名声衰落了。其实大多数人在去耶路撒冷的路上都会避开它。但是，神以他完美、恩典的计划，使用这个微不足道的小城将他的儿子带到了世上。那一天，伯利恒光芒万丈：“多年来的希望和恐惧，今晚在你身上得到了满足。

是的，那一天，神的恩典在伯利恒展现出来，让我们可以带着他的恩典走向世界。

**祷告：天父，就像你使用伯利恒这个不起眼的小城把你的儿子带到世上，求你也使用我把他的好消息带给他人。**
