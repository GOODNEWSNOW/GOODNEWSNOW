# /今日与恩典同行/

### 第144 05/24 礼拜二

>但以理知道这禁令盖了玉玺，就到自己家里(他楼上的窗户开向耶路撒冷），三次，双膝跪在他神面前，祷告感谢，与素常一样。——但以理书6:10

底部破了，你会怎么做？如果是个杂货袋，补救措施很简单。但是，如果是你的生活你的婚姻、财务或健康——跌破谷底呢？你又该怎么办？

信徒并不能免除生命的危机，我们也会跌破谷底。像其他人一样，我们也会面临令人崩溃的失望和不幸。

但当我们看先知但以理，就会发现他跌破谷底时，所做的就是持守对神的忠诚。但以理仍(旧跪下来信靠神。他知道，当我们跪下的时候，最能体会神的恩典。

**祷告：天父，我知道在这个堕落的世界里，生活充满了失望和不幸。但在困难的时候，求你帮助我效法但以理的榜样，每天来到你面前祷告和感恩。**
