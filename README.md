# /今日与恩典同行/

### 第162 06/11 礼拜六

>也不要将你们的肢体献给罪作不义的器具；倒要像从死里复活的人，将自己献给神，并将肢体作义的器具献给神。罪必不能作你们的主，因你们不在律法之下，乃在恩典之下。——罗马书6:13-14

我们很多人都竭力想在讨神喜悅的渴望气我们仍旧会犯罪的事实之间作调和。或许你也问过这个问题：“神啊，如果我仍旧不住犯罪，我会从你的恩典中坠落吗？”

使徒保罗在罗马书第6章中明确地说，我们绝不会失去神的恩典。但他也足够清楚地说，我们不能滥用神所赐予的恩典。

我们都会有失足跌倒的时候，而我们虽然悖逆，神还是爱我们。但是，为罪忧伤显明了圣灵在我们里面，神的恩典也在我们生命中工作。不仅如此，蒙福的确据也激励我们照着神的意思悔改，从而恢复与神的关系！

**祷告：主啊，谢谢你拯救我脱离罪的奴役。求你帮助我今日就像在基督里死而复活的人，将自己献给你。**
