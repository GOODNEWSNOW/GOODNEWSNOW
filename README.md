# /今日与恩典同行/

### 第124日 05/04 礼拜三

>况且我们的软弱有圣灵帮助，我们本不晓得当怎样祷告，只是圣灵亲自用说不出来的叹息替我们祷告。鉴察人心的，晓得圣灵的意思，因为圣灵照着神的旨意替圣徒祈求。——罗马书8:26-27

你想参与奇妙的探索吗？创造万物的神透过祷告来成就他的旨意。我们不需要知道所有答案，或是走在正路上，才能被神使用。我们以祷告来到神面前，让他提供答案，给我们指明正路。

明白圣灵在铸告中的奇妙工作是何等叫人喜乐，因为行善的能力可以借由我们而出，而且重担已经从我们身上卸下。

借着圣灵，神甚至可以使用我们的错误来生出益处，只要我们继续在祷告中寻求他，并渴望为他的尊贵和荣耀而活。

**祷告：主啊，我有时不知道该如何祈祷，也不知道该祈求什么。感谢你的圣灵，他按照你完美的旨意为我代求。**
