# /今日与恩典同行/

### 第123日 05/03 礼拜二

>只是那人没有遇见配偶帮助他。耶和华神使他沉睡，他就睡了；于是取下他的一条肋骨，又把肉合起来。耶和华神就用那人身上所取的肋骨造成一个女人，领她到那人跟前。——创世记2:20b-22

妻子们，作配偶的帮助者，意味着你要帮助你的丈夫成为神所要的，而不是你所要的，这一点很重要。妻子如果总是想把丈夫改造成她想要的样子，就表明她没有正确理解神的命令。

妻子这样做，其实是在保留自己的某一部分爱和奉献，直等丈夫反映出她改造出来的完美，她是在剥夺他们两人作为夫妻所享有的当下。太多的夫妻在紧张和冲突的关系中浪费了多年时间，而他们本可以彼此相爱与珍惜。

所以，如果你为人妻，想改变你的丈夫，请采用合乎圣经的方法。让神在今天把你改造成合适的帮助者，去爱你的丈夫，支持你的丈夫。

**祷告：天父，请赐予我恩典，让我每天都能爱我的丈夫，帮助他成为你所希望的样子，而不是我所希望的样子。**
