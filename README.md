# /今日与恩典同行/

### 第107日 04/17 主日

>两个人总比一个人好，因为二人劳碌同得美好的果效。若是跌倒，这人可以扶起他的同伴。 -传道书4:9-10a

属灵得胜没有捷径，但所幸也并不神秘。关键是以祷告的心与人联系、对人负责。在健康的基督徒关系中互相服侍，我们才得以在基督里成长。而明白这些有效的敬虔之路，则赋予我们基督徒生活的信心。

我们的宗教不是法术，满是神秘咒语、秘密握手和秘传暗号。我们乃是与他人一起寻求如何更好地去爱，好帮助我们按神的旨意持守和成长。

今天就找一个人同行，因为两个人总比一个人好…若是跌倒，这人可以扶起他的同伴。今天就在心里默想这句话吧！

**祷告：父啊，就像你总是与我同行，今天请你帮助我以爱心扶持他**
