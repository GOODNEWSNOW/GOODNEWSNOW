# /今日与恩典同行/

### 第182 07/01 礼拜五

>凡管教的事，当时不觉得快乐，反觉得愁苦；后来却为那经练过的人结出平安的果子，就是义。——希伯来书12:11

在《梦幻巴士》一书中，C.S.路易斯讲到一个男人因停在他肩上的红蜥蜴而困扰不已。这只奚落他的蜥蜴代表和叫我们挣扎的事。

一天，有位天使承诺帮他摆脱这个折磨人的讨厌鬼，他高兴不已。可当天使正要击杀蜥蜴的时候，那人突然生出一个念头。如果没了这只熟悉的动物，生活会是什么样子的？天使就提醒他：就是现在。你必须决定!

我们从这个故事中学到，要对付那些不是从神而来的事物是不容易的。但他的恩典使我们摆脱了阳挡我们的蜥蜴，得以自由经历基督所赐丰盛的生命。

**祷告：天父，求你帮助我甘心顺服你的管教，尽管那很痛苦。我知道它最终会在我生命中结出“平安的果子，就是义”。**
