# /今日与恩典同行/

### 第126日 05/06 礼拜五

>就为你们不住地感谢神。祷告的时候，常提到你们，求我们主耶稣基督的神，荣耀的父，将那赐人智慧和启示的灵赏给你们，使你们真知道他，并且照明你们心中的眼睛，使你们知道他的恩召有何等指望，他在圣徒中得的基业有何等丰盛的荣耀；并知道他向我们这信的人所显的能力是何等浩大。——以弗所书1:16-19

沿着家乡附近的密西西比河，我有一个最喜欢的散步地点。就在发电厂的后面，河水被改道成狭窄河道，以便转动两个巨大的涡轮机，为城市供电。

当河水涌入河道，汹涌翻腾，吞没途中的一切，我想起了创世时神的灵在覆盖大地的水面上移动的时刻。

与圣灵在创世时所展现的能力相比，这条河中的巨大力量不过是沧海一粟。同样的圣灵现在也住在我们相信基督的人里面，让我们能运用神爱和恩典的能力，有信心地生活。

**祷告：天父，感谢你赐下圣灵。求你帮助我把握住这巨大的能力，使我能够有信心地活在基督里。**
