# /今日与恩典同行/

### 第115日 04/25 礼拜一

>所以，自己以为站得稳的，须要谨慎，免得跌倒。你们所遇见的试探，无非是人所能受的。神是信实的，必不叫你们受试探过于所能受的；在受试探的时候，总要给你们开一条出路，叫你们能忍受得住。我所亲爱的弟兄啊，你们要逃避拜偶像的事。——哥林多前书10:12-14

当我们在信仰中坚持的时候，神呼召我们站稳...忍受...和逃避。前两个词表示用力量对抗试探，而最后一个词则是指逃离试探。

这两个概念都很关键，因为战胜试探不是简单地坐在沙发上，手拿巧克力，叫神让试探消失。神不会用魔法使我们脱离眼前的
罪。

但你可以从哥林多前书中得到安慰，因为神是信实的，他不会让你受试探过于所能受的。当你站稳、忍耐和逃避试探时，他必会开一条出路。

**祷告：主啊，当我这一周面对试探时，求你帮助我寻求你预备的逃生之路，使我能逃离危险！**
