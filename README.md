# /今日与恩典同行/

### 第100日 04/10 主日

>你们为主的缘故，要顺服人的一切制度，或是在上的君王，或是君王所派罚恶赏善的臣宰。 -彼得前书2:13-14

你有没有想过，凡基督徒都要顺服神放在他们之上的人？妻子要顺服丈夫，孩子要顺服父母，员工要顺服上司。而那些有杈柄的人则蒙召成为仆人式领袖，牧养那些在他们照顾下的人。

这样，每个人都必须献上自己的恩赐、权利或权柄来服侍他人，以建立神的国度。无私牺牲和甘心顺服是基督徒经验的共同点，并不会贬损个人价值。

所以，我们要喜爱天堂的价值观，喜爱那不愿压迫任何人的仁慈之神！

**祷告：天父，请帮助我记住，每个人都必须服从于某人。求你赐给我力量，使我喜爱你的价值观，不要抗拒它们。**
