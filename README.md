_第28日 01/28_

> 神却拣选了世上愚拙的，叫有智慧的羞愧；又拣选了世上软弱的，叫那强壮的羞愧。神也拣选了世上卑贱的，被人厌恶的，以及那无有的，为要废掉那有的。使一切有血气的，在神面前一个也不能自夸。 -哥林多前书1:27-29

有一个古老的故事是这么说的：当耶稣升天后，他召集了天军，告诉他们自己接下来要做什么。他说，他会使用他所拯救的软弱、有罪和污秽之人，用他们作活石来建立他子民的根基。那是要改变世界的一群子民！

下面鸦雀无声，天使加百列打破了沉默，问道：“那第二方案是什么？”可是并没有第二方案，因为神的计划向来都是使用这样的人来建立恩典的根基，好在世上建立盼望。

今天你要大得激励，因为知道自己也是耶稣子民的一分子，他要在其上建立这世界的盼望！

###### 祷告：天父，我很受激励，因为你拣选了像我这样的人来成就你在世上的工作。今日请赐予我恩典来给他人带去好的影响。

_第27日 01/27_

> 因为我们作仇敌的时候，且借着神儿子的死，得与神和好；既已和好，就更要因他的生得救了。 -罗马书5:10

我们很多人都对自己的孩子说过这样的话：“如果你做个好孩子，耶稣就会爱你。”这听上去很有爱，却与福音背道而驰。让小孩因为愧疚而听话，告诉他们听话神就会爱他们，这实在是属灵毒药。

耶稣从来没有因为他哪个孩子好而爱他，无论老少。耶稣爱他的孩子是因为他的良善。孩子如果为了得到耶稣的爱而表现得听从耶稣，那么他长大了就会怀疑耶稣的爱——尤其是当生命中的挑战清楚表明，我们不可能永远是乖小孩。

耶稣真的爱你，不是因为你的好，而是因为他的良善！

###### 祷告：耶稣，谢谢你爱我不是因为我是好人，而因为你是良善、恩慈和怜悯的。求你帮助我今天就去爱别人，回应你的恩典。

_第26日 01/26_

> 感谢神！因为你们从前虽然作罪的奴仆，现今却从心里顺服了所传给你们道理的模范。你们既从罪里得了释放，就作了义的奴仆。 - 罗马书6:17-18

我们心里可能会这样盘算：“如果神最终会以基督的善行替代我的坏行为，那我还是继续犯罪吧！”可是我们这样想对吗？使徒保罗在罗马书第6章驳斥了这种暗昧的盘算，他告诉我们：“我们可以仍在罪中，叫恩典显多吗？断乎不可！”

保罗警戒我们不要轻忽神的恩典和怜悯，但他也从未削弱神恩典的福音。因为我们是因信称义，不是靠我们的行为——因此没人能在基督之外夸口！

我们不应当在罪或愧疚中活着；我们应承认自己的罪，安息在神的恩典和怜悯中！

###### 祷告：父啊，谢谢你释放我脱离罪的奴役，使我成为义的奴仆。愿我永不妄用你的恩典，而是以感恩与顺服回应。

_第25日 01/25_

> 他们说：“来吧！我们要建造一座城和一座塔，塔顶通天，为要传扬我们的名，免得我们分散在全地上。” - 创世记11:4

除了基督教，所有宗教都在讲人必须做些什么才能接近神，倡导身体或精神上要付出努力。但圣经教导我们，精神或身体的操练毫无用处。根本没有什么天梯！

所以巴别塔的故事在圣经里很早就发生了。制造我们自己的“天梯”不是通往神的途径。基督教信息的独特之处就在于，神到我们这里来是为了使我们成为属他的人。

要安稳在这个圣经真理中，晓得既然我们永远没法靠自己接近他，我们满有恩典的天父就顺着梯子往下走向我们！

###### 祷告：主啊，我知道我没法靠自己的努力上到你那里。谢谢你借着耶稣基督降临到地上来拯救我们。

_第24日 01/24_

> 所以，我们只管坦然无惧地来到施恩的宝座前，为要得怜恤，蒙恩惠，作随时的帮助。 - 希伯来书4:16

有很多年，我都以为我越公义，神就会越喜爱我。当我读到罗马书12:1这样的经文，“所以弟兄们，我以神的慈悲劝你们，将身体献上，当作活祭，是圣洁的，是神所喜悦的；你们如此侍奉乃是理所当然的”，我以为圣经也证实了我的想法。

但我没领悟到这段经文中的词组“以神的慈悲”。因神的慈悲我们才成为圣洁、可喜悦的。

所以请记住，我们的圣洁和蒙神喜悦不是我们自己的功劳，而是他的！因基督的恩典我们才能在神面前称义，我们可以为他的怜悯胜过审判而欢喜！

###### 祷告：父啊，谢谢你以慈悲待我！我知道我是个蒙你恩典拯救的罪人，而不是靠我自己的功劳。

_第23日 01/23_

> 亚波罗算什么？保罗算什么？无非是执事，照主所赐给他们各人的，引导你们相信。我栽种了，亚波罗浇灌了，惟有神叫他生长。可见栽种的，算不得什么，浇灌的，也算不得什么；只在那叫他生长的神。 - 哥林多前书3:5-7

如果只讲大卫战胜歌利亚而不提他与拔示巴犯的罪，或者不讲他谋杀拔示巴丈夫乌利亚的罪，那样算是诚实吗？如果我们只鼓励人们认识大卫好的一面，那我们教导的是真理吗？

真理就是，圣经只有一位英雄-神自己。其他人都只是故事中的配角。关注英雄人物而无视神的恩典，就会让他们的英雄事迹扭曲福音的信息。神没有等到新约才展现他的恩典。恩典出现在圣经的每一页！

所以，下次你读圣经的时候，要去找那个真正的大英雄和他恩典的福音.

###### 祷告：主啊，求你帮助我看见你是圣经每一个故事的真正英雄。愿我在所读的每一页都看见你恩典的光芒。

_第22日 01/22_

> 耶和华啊，愿你照你的话，使你的慈爱，就是你的救恩，临到我身上，我就有话回答那羞辱我的，因我倚靠你的话。诗篇119:41-42

当你打开圣经，你看见的是什么？满是责任和教义？还是一种关系，这关系以圣经每一页都有的恩典与爱为本？

假如我们打开圣经是要看各种规范条例，学习如何表现得更好、如何赚取某人的爱与恩惠，那这信息里的盼望在哪里呢？

我们打开圣经的时候应该明白，那其实是一封精心构思的情书。那样，我们就会看见贯串全书的爱的主线——亚当和夏娃犯罪后神如何在园中与他们相遇，今日神又如何以他的慈爱和恩典与我们这些堕落的人相遇！

###### 祷告：天父，求你帮助我在圣经每一页看见你的爱与眷顾。愿这能激励我向别人显出你的爱。

_第21日 01/21_

> 你们所遇见的试探，无非是人所能受的。神是信实的，必不叫你们受试探过于所能受的；在受试探的时候，总要给你们开一条出路，叫你们能忍受得住。 - 哥林多前书10:13

我们都面对过年少轻狂的试探，自以为战无不胜，也表现得好像自己战无不胜。但是，就像使徒保罗指出的：“你们所遇见的试探，无非是人所能受的。”

这是我们的困境，我们的挣扎。我们都很脆弱。可是我们并不孤单，这应该让我们得安慰。我们每个人心里都有嫉妒、愤怒和贪梦的种子。如果我们承认这一点，就应该更竭力寻求神的赦免和帮助，他满有恩典地赐下这些，因为他知道我们不过是人。

下一次你受试探的时候要感到安慰，因为你并不孤单。祈求神所应许的恩典，你就会得着力量。

###### 祷告：主啊，当我今日面对试探，愿你赐下逃脱之路，好叫我不得罪你

_第20日 01/20_

> 从他丰满的恩典里，我们都领受了，而且恩上加恩。律法本是借着摩西传的；恩典和真理都是由耶稣基督来的。- 约翰福音1:16-17

生活有时像大富翁游戏一样。你或是抵达公园广场，赢了游戏，或是抽到了可怕的“坐牢卡”，前往“监狱”。你做得好不好取決于这个游戏你玩得好不好。

但现实是，玩好生命这场游戏并不会送你上天堂，也不会使你免下地狱。这个真相改变了我们的游戏规则。这是不是意味着我们的行为对神而言无关紧要？

当然不是。它的意思是，我们的好行为不应当由预设的奖赏或可怕的后果所驱动。我们的回应不应当出自愧疚或得利的念想，而是因为感激神的恩典。这样的恩典既不是我们赚来的，也不是我们配得的，而要凭信心领受，并以充满爱的生命表露出来。

###### 祷告：父啊，谢谢你如此爱我，竟差遣你的儿子为我的罪而死。愿我以恩待他人来回应你的恩典。

_第19日 01/19_

>凡遵守他的约和他法度的人，耶和华都以慈爱诚实待他。 -诗篇25:10

我们的国家公园和森林里都有规划好的路线，以保证徒步者、骑行者和露营者人身安全。同样，神律法的原则也为神的子民设定了安全路线。所以，行走在这样的路上我们无需惧怕，而要以之为引导和保障。

这路并不能为我们赚取神的恩典。它体现了神恩慈的眷顾，也表达了诗人的喜悦，他写道：“我何等爱慕你的律法，终日不住地思想。”（诗119:97） 弃绝神安稳的善道，并不是行在自由中，而是行在愚味中。

今天就行在神恩慈供应的安稳善道之上，你会更多发现神的性情和眷顾！

###### 祷告：天父，你的话照亮了我每日当行的道路。求你帮助我今天行走在你预备的这条安稳之路上。

我住在林肯的故乡，就是伊利诺伊州中部的一个地区。这里关于第16任总统的传说和事实相互交织，有时很难分辨孰真孰假。但有关他生命的故事反映的总是他所遵行的原则。

有人说林肯曾用他微薄的工资在奴隶拍卖会上出最高价买了一个奴隶。买了那奴隶后，他就把她释放了，并对她说，她想去哪儿就可以去哪儿。她回答他说：“我想跟你走！，这样的忠诚来自于脱离奴役而生出的感恩。因此，我们今天也要因为神使我们脱离罪的捆绑而向他献上忠诚和感谢！

###### 祷告：：主啊，谢谢你除掉我的捆绑，使我在基督里得自由。请帮助我每天都跟随你！

_第17日 01/17_

>神却拣选了世上愚拙的，叫有智慧的羞愧；又拣选了世上软弱的，叫那强壮的羞愧。神也拣选了世上卑贱的，被人厌恶的，以及那无有的，为要废掉那有的，使一切有血气的，在神面前一个也不能自夸。- 哥林多前书1:27-29

我们都会对那些世界级的音乐家、演员和运动员赞叹不已。这些成绩斐然的超级明星往往不断受媒体关注，收入也远超出天赋能力不及他们的人叫人难过的是，这种以表现为本的心态可能会深深影响我们的自我认同，使我们以为，我们只要表现得好就会有人爱。

没错，神看重我们的顺服，但那不是他看顾我们的原因。神看顾我们是因为耶稣为我们所做的，而且我们是属耶稣的。

因此，以平静而有力量的心服侍神，知道耶稣的表现而非你的表现，已经决定了你与神的关系！

###### 祷告：天父，我知道你接纳我不是因为我的表现，而是因为耶稣在十字架上所做的。求你帮助我以他的公义为乐！
