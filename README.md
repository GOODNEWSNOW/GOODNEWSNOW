# /今日与恩典同行/

### 第148 05/28 礼拜六

>所以你们要彼此认罪，互相代求，使你们可以得医治。义人祈祷所发的力量是大有功效的。——雅各书5:16

(Jonathan Edwards）牧师召集了八百个男人一起祈祷。而就在这次聚会中，一个女人给他们传了一条信息，请他们为她的丈夫祷告。纸条上描述了这个男人，他属灵的骄傲已使他变得没有爱心、骄傲又难以相处。爱德华兹希望他朗读信息的时候，这个男人能举起手来，接受众人的祈祷。

结果，爱德华兹震惊了。有三百人因圣灵而知罪，举起了手！这个经历告诉我们，那些有悔改之心的人被神的恩典所充满，他们渴望承认自己的罪，与神和他人恢复正确的关系！

**祷告：耶稣，我承认我还不是你所期望我成为的那个人。感谢你的救免和圣灵在我生命中的工作。**
