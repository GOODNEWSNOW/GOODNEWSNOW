# /今日与恩典同行/

### 第103日 04/13 礼拜三

>我的肺腑是你所造的；我在母腹中，你已覆庇我。我要称谢你，因我受造，奇妙可畏。-诗篇139:13-14b

神用美妙的创造力，使每个孩子都与众不同。我们不要因为自己的孩子与人不同就感到沮丧或困惑。神的创造力赋予每个人尊严，父母因此要照着主的旨意调整自己的心。所以说，为人父母必须要与神保持爱的关系；身为基督徒父母，我们的目标就是反映我们的救主。

每一次我们勒住舌头，控制怒气，忍受被误解，花时间和孩子——起打球，承受侮辱，耐心地爱，永远饶恕，我们都在反映基督的爱，并把我们的孩子引向救主。

所以，今天就让你的孩子因你而明白神的爱和恩典吧！

**祷告：天父，谢谢你美妙的创造力！求你帮助我因我的孩子与众不同而欢喜快乐，并向他们反映你的爱和恩典。**
