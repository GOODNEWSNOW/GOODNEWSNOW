# /今日与恩典同行/

### 第97日 04/07 礼拜四

>因为我深信无论是死，是生，是天使，是掌权的，是有能的，是现在的事，是将来的事，是高处的，是低处的，是别的受造之物，都不能叫我们与神的爱隔绝；这爱是是在我们的主基督耶稣里的。-罗马书8:38-39

神知道我们可能会怀疑他的爱。所以，在新约极为复杂的•段经文中，神让使徒保罗回到了这些信件核心的简单主题：神无条件的爱

在这段经文中，你几乎可以听到神对保罗低声说：“保罗，这对我的子民来说很难理解，但为了叫他们不灰心，请提醒他们，耶稣爱他们，没有什么能改变这一点。

因此，保罗在罗马书中写道，没有什么能使我们与神在基督耶稣里的爱隔绝-罪不能，环境不能，属灵仇敌也不能。无论你的生命正处于何境，如果你已经认耶稣作你的救主，今天和每一天，你都可以这样确信！

**祷告：主啊，你的爱永不改变，我很难领会这一点。今天，求你帮助我思想你在基督里赐给我的确据。**
