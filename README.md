# /今日与恩典同行/

### 第95日 04/05 礼拜二

>你们中间也有人从前是这样；但如今你们奉主耶稣基督的名，并借着我们神的灵，已经洗净、成圣、称义了。 -哥林多前书6:11

在科罗拉多州的大沙丘国家公园，沙丘高达213米以上。沙丘的高度很壮观，不过它们的形成过程也有些奇妙。

令人惊奇的是，梅达诺河将沙丘底部无数的沙子冲进下面的山谷。你肯定以为水会侵蚀沙丘，至少会让沙丘变小。然而，河水冲刷而下的沙子沉积在两座山脉之间的山谷，风不断地将沙吹回沙丘使沙丘越堆越高！

同样，神洗去我们的罪，不是要减损我们的重要性，而是为了把我们朝着他的方向建造得越来越高！

**祷告：父啊，感谢你永远洗去我的罪孽。当你把我朝着你的方向越造越高的时候，求你帮助我每天反映你的圣洁品格。**
