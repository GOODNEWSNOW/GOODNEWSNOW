# /今日与恩典同行/

### 第163 06/12 主日

>既是这样，还有什么说的呢？神若帮助我们，谁能敌挡我们呢？神既不爱惜自己的儿子，为我们众人舍了，岂不也把万物和他一同白白地赐给我们吗？——罗马书8:31-32

整本圣经，处处都有神的恩典。恩典使我们能与神建立关系，尽管我们有这样那样的不足。神在耶稣基督里赦免我们，尽管我们的悔改很不够。恩典将我们犯罪的后果过滤，使我们在灵性上不受伤害。

所以，当我们失足跌倒，我们可以满有信心地倚靠神无限的恩典。正如使徒保罗在罗马书8:34所写：“谁能定他们的罪呢？有基督耶稣已经死了，而且从死里复活，现今在神的右边，也替我们祈求。”

因此，不要理会定罪的声音，而要稳稳站立于神恩典的真理之上以及基督在十字架上为遮盖你一切的罪已经做成的工！

**祷告：天父，谢谢你在耶稣基督里丰盛的恩典遮盖了我一切的罪！愿这恩典激励我更好地为你而活。**
