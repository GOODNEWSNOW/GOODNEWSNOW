# /今日与恩典同行/

### 第120日 04/30 礼拜六

>你们作丈夫的，要爱你们的妻子，正如基督爱教会，为教会舍己。要用水借着道把教会洗净，成为圣洁，可以献给自己，作个荣耀的教会，毫无玷污、皱纹等类的病，乃是圣洁没有瑕疵的。——以弗所书5:25-27

你有没有想过，文夫当如何活出神的命令，为妻子舍己？这当然很有难度。只有当一个男人从神那里汲取恩典作他的最终资源时，才有可能做到。靠着基督爱的泉源，丈夫才能为妻子舍己。

使徒保罗说：“这是极大的奥秘。”所言甚是！但神给我们指明了道路，如何能将这个奥秘变为日常实践。圣经中丈夫对妻子的舍己之爱，其根基和源头在于领会基督的爱——仅此而已。所以，作丈夫的，你若每天与基督会面，就会得着能力去爱你的妻子、服侍你的妻子。

**祷告：主啊，帮助我每天从你那里汲取恩典和力量，这样我就能像你爱教会那样爱我的妻子了。**
