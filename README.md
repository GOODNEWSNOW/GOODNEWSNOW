# /今日与恩典同行/

### 第176 06/25 礼拜六

>法利赛人站着，自言自语地祷告说：“神啊，我感谢你，我不像别人勒索、不义、奸淫，也不像这个税吏……”那税吏远远地站着，连举目望天也不敢，只捶着胸说：“神啊，开恩可怜我这个罪人！”  ——路加福音18:11、 13

“罪名成立！”这四个字是我们绝对不想听到的，特别是事关我们在救主耶稣可怕死亡中的角色时。我们不肯认为自己有分于送他上十字架的罪行，因为把矛头指向犹大、彼拉多和公会这样的关键角色要容易得多。但我们是有分于这罪的，正因此故，耶稣才为我们舍命。

耶稣的十字架是恩典的终极面面，因为借着神的奇昇恩典，耶稣担当了你我的罪，好叫我们被称为义。无瑕疵的羔羊、无罪的万王之王为我们罪孽的后果付上了最终的代价，并使我们能进入他无尽的恩典。

**祷告：天父，我知道我悖逆得罪了你。谢谢你差遣你的儿子承担我本当受的刑罚！**
