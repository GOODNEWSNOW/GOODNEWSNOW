# /今日与恩典同行/

### 第169 06/18 礼拜六

>我们众人既然敞着脸得以看见主的荣光，好像从镜子里返照，就变成主的形状，荣上加荣，如同从主的灵变成的。——哥林多后书3：18

假如我们足够诚实，都会痛苦地意识到自己的软弱和缺陷。那我们怎样才能达到神对圣洁的要求呢？约翰•班扬经典小说 《天路历程》 中那位有名的天路客就描绘了一个答案。

在旅程尾段，天路客发现了一面神奇的镜子。镜子的正面没什么特别，背面则是耶稣被钉十字架的形象。你只要往镜子里看，就能看到自己人性中常有的瑕疵和伤痕。可是当看背面，你看到的就只有神儿子的荣耀。这个故事的镜子就回答了我们怎样才能在今生成为圣洁——不是靠你自己的功劳，而是要反映神在耶稣基督里的恩典。

**祷告：父啊，当我今天在耶稣基督面前注目你的荣耀，愿我更多地转变成他的形象。**
