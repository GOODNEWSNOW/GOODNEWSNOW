# /今日与恩典同行/

### 第116日 04/26 礼拜二

>你们虽然没有见过他，却是爱他；如今虽不得看见，却因信他就有说不出来、满有荣光的大喜乐；并且得着你们信心的果效，就是灵魂的救恩。——彼得前书1:8-9

福音的信息必须反映在神子民的态度上，因为我们所饮的活水会影响那些我们与之分享基督的人。

基督的好消息从不轻视罪的严重性，也不会掩盖神的大能和赦免何等奇妙。认罪的眼泪加上救免带来的喜乐，会产生感恩的心，使被改变的人有能力过基督徒的生活。

当我们真正领会赦免我们的那一位，他的心是多么伟大，我们的心就开始与他的心和谐跳动。今日请荣耀主，迎接神的喜乐、爱和恩典，这能改变你和你周国人的生命。

**祷告：主啊，求你帮助我今天荣耀你，记念你在耶稣基督里的大能和赦免何等奇妙！**
