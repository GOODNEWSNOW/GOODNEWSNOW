# /今日与恩典同行/
 
### 第112日 04/22 礼拜五

>惟有那以力量束我的腰、使我行为完全的，他是神。他使我的脚快如母鹿的蹄，又使我在高处安稳。 -诗篇18:32-33

我们一家人喜欢去科罗拉多州度假。我们经常会去参观一座石头城堡，通过螺旋楼梯爬上去，在山顶观看壮观又险峻的景色。

我儿子还小的时候，即使那里没有栏杆，也会和我一起爬楼梯。在低层，我必须提醒他们拉住我的手，但越往上，他们越不需要提醒。到了山顶，我感觉自己的手仿佛被夹在两只小老虎钳里！

我儿子知道，越往高处就越危险神要把你带到高处，你可以心甘情愿、毫无畏惧地前往，因为他牵着你的手！

**祷告：父啊，当你带领我在基督里走向更高之处，请帮助我心甘情愿、毫无畏惧地前往，因为知道你与我同在。**
