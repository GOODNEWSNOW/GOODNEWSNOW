_第20日 01/20_

> 从他丰满的恩典里，我们都领受了，而且恩上加恩。律法本是借着摩西传的；恩典和真理都是由耶稣基督来的。- 约翰福音1:16-17

生活有时像大富翁游戏一样。你或是抵达公园广场，赢了游戏，或是抽到了可怕的“坐牢卡”，前往“监狱”。你做得好不好取決于这个游戏你玩得好不好。但现实是，玩好生命这场游戏并不会送你上天堂，也不
会使你免下地狱。

这个真相改变了我们的游戏规则。这是不是意味着我们的行为对神而言无关紧要？当然不是。它的意思是，我们的好行为不应当由预设的奖赏或可怕的后果所驱动。我们的回应不应当出自愧疚或得利的念想，而是因为感激神的恩典。这样的恩典既不是我们赚来的，也不是我们配得的，而要凭信心领受，并以充满爱的生命表露出来。

###### 祷告：父啊，谢谢你如此爱我，竟差遣你的儿子为我的罪而死。愿我以恩待他人来回应你的恩典。

_第19日 01/19_

>凡遵守他的约和他法度的人，耶和华都以慈爱诚实待他。 -诗篇25:10

我们的国家公园和森林里都有规划好的路线，以保证徒步者、骑行者和露营者人身安全。同样，神律法的原则也为神的子民设定了安全路线。所以，行走在这样的路上我们无需惧怕，而要以之为引导和保障。

这路并不能为我们赚取神的恩典。它体现了神恩慈的眷顾，也表达了诗人的喜悦，他写道：“我何等爱慕你的律法，终日不住地思想。”（诗119:97） 弃绝神安稳的善道，并不是行在自由中，而是行在愚味中。

今天就行在神恩慈供应的安稳善道之上，你会更多发现神的性情和眷顾！

###### 祷告：天父，你的话照亮了我每日当行的道路。求你帮助我今天行走在你预备的这条安稳之路上。

_第18日 01/18_

>我实实在在地告诉你们，所有犯罪的就是罪的奴仆。奴仆不能永远住在家里;儿子是永远住在家里。所以天父的儿子若叫你们自由，你们就真自由了。 - 约翰福音8:34-36

我住在林肯的故乡，就是伊利诺伊州中部的一个地区。这里关于第16任总统的传说和事实相互交织，有时很难分辨孰真孰假。但有关他生命的故事反映的总是他所遵行的原则。

有人说林肯曾用他微薄的工资在奴隶拍卖会上出最高价买了一个奴隶。买了那奴隶后，他就把她释放了，并对她说，她想去哪儿就可以去哪儿。她回答他说：“我想跟你走！，这样的忠诚来自于脱离奴役而生出的感恩。因此，我们今天也要因为神使我们脱离罪的捆绑而向他献上忠诚和感谢！

###### 祷告：：主啊，谢谢你除掉我的捆绑，使我在基督里得自由。请帮助我每天都跟随你！

_第17日 01/17_

>神却拣选了世上愚拙的，叫有智慧的羞愧；又拣选了世上软弱的，叫那强壮的羞愧。神也拣选了世上卑贱的，被人厌恶的，以及那无有的，为要废掉那有的，使一切有血气的，在神面前一个也不能自夸。- 哥林多前书1:27-29

我们都会对那些世界级的音乐家、演员和运动员赞叹不已。这些成绩斐然的超级明星往往不断受媒体关注，收入也远超出天赋能力不及他们的人叫人难过的是，这种以表现为本的心态可能会深深影响我们的自我认同，使我们以为，我们只要表现得好就会有人爱。

没错，神看重我们的顺服，但那不是他看顾我们的原因。神看顾我们是因为耶稣为我们所做的，而且我们是属耶稣的。

因此，以平静而有力量的心服侍神，知道耶稣的表现而非你的表现，已经决定了你与神的关系！

###### 祷告：天父，我知道你接纳我不是因为我的表现，而是因为耶稣在十字架上所做的。求你帮助我以他的公义为乐！
