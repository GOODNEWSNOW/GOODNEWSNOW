# /今日与恩典同行/

### 第174 06/23 礼拜四

>不但如此，就是在患难中也是欢欢喜喜的；因为知道患难生忍耐，忍耐生老练，老练生盼望；盼望不至于羞耻，因为所赐给我们的圣灵将神的爱浇灌在我们心里。——罗马书5:3-5

每当我们面临挑战，很自然就会问，神是否离弃了我。神为什么不让忠信人的工作轻省一点？一个原因，逆境是为了预备我们打前头的属灵之战。先知但以理帮助我们明白神没有离弃我们，只是在预备我们做更大的工。

有一天我偶然听到教会一位敬虔的姐妹说：“我在想，主要预备布莱恩做什么，才会让他经历如此大的试炼？”这句话有些吓人，使我好奇摆在前面的会是什么，但也帮助我认识到，试炼的时刻并非毫无目的。神借着他的恩典预备我们，而不是离弃我们。

**祷告：主啊，当我面临生命的挑战，求你帮助我认识到，这些挑战是要帮助我在基督里更有力量，预备我结出更硕大的果实。**
