# /今日与恩典同行/

### 第165 06/14 礼拜二

>人不可自欺。你们中间若有人在这世界自以为有智慧，倒不如变作愚拙，好成为有智慧的。——哥林多前书3：18

法国数学天才帕斯卡多年来一直在逃离神，就像当时和今日许多人一样。可是在1654年11月，帕斯卡遇见了神那改变生命的恩典。

帕斯卡去世八年后，有人发现了他外套里缝的一张羊皮纸条，上面记录了他那充满能力的归信：恩典之年1654年11月23日，星期一……大约晚上十点半到午夜三十分钟，火。亚伯拉罕、以撒、雅各的神，不是哲学家和学者的神，确信，发自内心的喜乐，平安。耶稣基督的神。耶稣基督的神。“我的神和你的神”……喜乐，喜乐，喜乐，喜乐的泪水……耶稣基督。耶稣基督。愿我与他永不分离。

你有过这样与神相遇的经历吗？就像帕斯卡样，你可以邀请耶稣基督成为你的救主，借着他的恩典改变你！

**祷告：父啊，当我从我救主耶稣基督的面容瞻仰你的荣耀，愿我被你的恩典改变！**
