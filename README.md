# /今日与恩典同行/

### 第140 05/20 礼拜五

>亚伯拉罕的后裔，大卫的子孙，耶稣基督的家谱：犹大从她玛氏生法勒斯和谢拉…撒门从喇合氏生波阿斯；波阿斯从路得氏生俄备得；俄备得生耶西；耶西生大卫王。大卫从乌利亚的妻子生所罗门。——马太福音1:1-6

耶稣的家世和你预想的并不一样。例如，有被恶待、抛弃又遭受不公指控的她玛，还有遭人鄙夷的妓女喇合，还有来自摩押的外邦人路得。还有与大卫王犯奸淫、大卫所杀之人的遗孀拔示巴。耶稣的家谱中包含了一些很不完美的人物。

但神借着基督的家谱告诉了我们一些重要真理。他在提醒我们，他并不惊讶于我们的脏衣服，不完美的人正是承受他恩典的最佳人选。神喜悦我们，不是因为我们做了什么，而是因为基督的良善。

神给我们穿上基督的公义，让他的光亮穿透污秽和黑暗，这不是很奇妙吗！

**祷告：天父，感谢你接纳甚至使用像我这样堕落的罪人。求你今日帮助我在你的恩典和良善中欢喜。**
