# /今日与恩典同行/

### 第104日 04/14 礼拜四

>无论做什么，或说话或行事，都要奉主耶稣的名，借着他感谢父神。 -歌罗西书3:17

我父亲在南方谋生养家，他教每个儿子如何使用横锯。在秋风习习的早晨，我们锯断了一块里面已经腐烂的木头，木头就从架子上掉下来，砸到地上，裂开了，儿时的我以为那是马头的形状。

于是，我把那个马头收起来，加上了其他一些零件，在父亲生日时把这最精致的马形领带架送给了他。他爱不释手地用了好几年-虽然它并不像我曾经以为的那样灵巧，做工也不精细。我父亲用它不是因为它的好，而是因为他的好。

同样，我们的天父接受我们的礼物，不是因为它们值得他爱：而是因为他就是爱！

**祷告：父啊，我把自己和今天所做的工作当作礼物献给你。谢谢你慈爱地接受我微不足道的献上。**
