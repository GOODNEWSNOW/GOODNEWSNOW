# /今日与恩典同行/

### 第117日 04/27 礼拜三

>因为神赐给我们，不是胆怯的心，乃是刚强、仁爱、谨守的心。你不要以给我们的主作见证为耻，也不要以我这为主被囚的为耻；总要按神的能力，与我为福音同受苦难。神救了我们，以圣召召我们，不是按我们的行为，乃是按他的旨意和恩典。 ——提摩太前书2:7-9a

宣教士克劳斯 （Myron Klaus）在哥斯达黎加与最贫穷的人相处多年。他的事工重点是那些住在垃圾堆，从中搜寻食物的城市孤儿。

教育这些儿童使其生存下来是一项艰巨的任务。但有个办法很奏效，就是训练他们如何公开演讲，让他们参加当地的比赛。

有一个男孩表现出色，赢得了全国冠军，并获邀向总统及其夫人演讲。这个男孩面向哥斯达黎加总统讲话时满有自信，流露出他对总统和他救主的喜爱之情。

恩典何等伟大，它激发一个小男孩向人述说，因为神无条件的爱，我们可以毫无惧怕地走在他的道路上！

**祷告：天父，愿你的恩典激励我大胆说出你在耶稣基督里无条件的爱**
