# /今日与恩典同行/

### 第151 05/31 礼拜二

>我将这些事告诉你们，是要叫你们在我里面有平安。在世上，你们有苦难；但你们可以放心，我已经胜了世界。——约翰福音16:33

当安（Ann Judson）陪着她丈夫阿多尼兰(Adoniram Judson）去印度时，她没有想到自己在向数百万人传福音的事工中竟如此重要。她也绝对想不到她的功用需要怎样的顺服才得以发挥。

当英国和印度之间关系紧张时，传教士常遭监禁。安的丈夫就是如此，他靠她的话活了下来：“不要放弃，神会使我们得胜。”当希望在别人身上消失殆尽的时侯，安经常重复的话语让阿多尼兰活了下来。当安从今生进入到荣耀里的时候，阿多尼兰把她的话当成了神的命令要继续分享恩典福音的好消息。就这样，数百万人，就是安的属灵儿女，因着一个女人对救主的爱而诞生。

**祷告：父啊，感谢你赐给我们在基督里得胜的应许。求你帮助我与那些不认识他的人分享这个好消息。**
