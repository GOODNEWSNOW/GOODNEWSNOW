# /今日与恩典同行/

### 第167 06/16 礼拜四

>只是在你们中间，不可这样；你们中间谁愿为大，就必作你们的用人；谁愿为首，就必作你们的仆人。正如人子来，不是要受人的服侍，乃是要服侍人，并且要舍命，作多人的赎价。——马太福音20:26-28

你有没有留意到，许多人来教会时心里都有个清单？他们会问：“牧师的讲道是我喜欢的吗？音乐符合我的品味吗？教会建筑有合宜的氛围和装饰吗？”但这不正错失了教会存在的本意吗？亦或错失了跟随基督
的本意？

耶稣在马太福音16:24说：“若有人要跟从我，就当舍己。”这不止是呼召我们偶尔行善，而是命令我们要把神所看重的事放在自己看重的事之上，像耶稣那样服侍他人。如果跟随基督，你就会发现你最大的喜乐就是照神满有恩典的计划而活，去服侍他人而不是受人服侍。

**祷告：耶稣，谢谢你来不是要受人的服侍，乃是要服侍人。求你帮助我追随你的榜样，情愿献上生命服侍他人。**
