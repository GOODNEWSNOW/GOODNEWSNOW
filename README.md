# /今日与恩典同行/

### 第160 06/09 礼拜四

>我要教导你，指示你当行的路；我要定睛在你身上劝戒你。你不可像那无知的骡马，必用嚼环辔头勒住它；不然，就不能驯服。——诗篇32:8-9

每年我们很多人都会设立目标，写下新年立志，为要使自己朝着所期待的奖赏前行。设立目标完全没有问题，只是如果我们想要在神的恩典之外提升自己，就会失去他为我们所定的平安。

我们往往会选择那些能产生外在结果的目标，但神的旨意是要让我们的内在也闪闪发光！神的目标满有恩典和慈爱，使我们能持守在满有平安和喜乐的道路上，只要我们谨守他在圣经上的话语。在圣经里我们看见他的恩典彰显，也学习去寻求最终的奖赏我们的救主耶稣基督。

**祷告：主啊，愿你的话语使我持守正道，追求你为我设立的目标。求你帮助我寻求耶稣基督，他是那最终的奖赏。**
