# /今日与恩典同行/

### 第122日 05/02 礼拜一

>你们作妻子的，当顺服自己的丈夫，如同顺服主。因为丈夫是妻子的头，如同基督是教会的头；他又是教会全体的救主。教会怎样顺服基督，妻子也要怎样凡事顺服丈夫。——以弗所书5:22-24

使徒保罗告诉我们，妻子要“凡事”顺服丈夫。但这并不是说，妻子要对丈夫言听计从，不顾道德后果。

妻子要顺服丈夫，就像教会顺服基督一样，因此，她仍要在属灵上对自己的行为负责。丈夫无权要求妻子做任何违背神话语的事。顺服丈夫永远不会高于顺服神。因为顺服是一种敬拜的行动，其主要目的是为了荣耀神。

这就界定了我们在婚姻中要如何彼此服侍，也提醒我们，我们服侍别人不是因为他们配得，而是因为神配得！

**祷告：主啊，我知道我顺服别人最终表示我顺服你。求你帮助我每夭都这样荣耀你。**
