# /今日与恩典同行/

### 第168 06/17 礼拜五

>你们作丈夫的，也要按情理和妻子同住；因她比你软弱，与你一同承受生命之恩的，所以要敬重她。这样，便叫你们的祷告没有阻碍。——彼得前书3:7

随着我们孩子的年龄、所需的开支以及我工作要求的变化，我妻子凯茜不断调整她出门工作所花的时问和精力。她一直坚持将神所托付给她照看之人的需要摆在她自己的益处之上。在这样一个混乱的世代，许多人都会质疑她的选择，但是我尊重她的选择和她。

使徒彼得说：“你们作丈夫的，也要按情理和妻子同住…要敬重她。”这些话提醒我要告诉我妻子，我十分敬重她为全家的益处而付出的牺牲。每一位看重神所看重之事的妻子都理当听见这话。天国也尊荣凡支持她丈夫和家庭的妇人，她体现了神持续的恩典。

**祷告：天父，求你帮助我告诉我的配偶，我十分敬重他（她）为我们全家的益处所做的牺牲。**
