# /今日与恩典同行/

### _第95日 04/05 礼拜二_

>你们来到主面前，也就像活石，被建造成为灵宫，作圣洁的祭司，借着耶稣基督奉献神所悦纳的灵祭。-彼得前书2:5

我们社区有新的开发项目开始施工时，建筑商都会在物业四周安上栅栏，只允许有助于开发新房的物资进入门内。

同样，神也在我们的生命周国安了一道栅栏，只允许能使我们更像耶稣的东西入内。

神眷顾的栅栏是基督徒最大的安慰之一。它向我们保证，只有对我们有终极益处的东西才能进入我们的生命。这并不表示万事如意，却意味着我们所遇到的一切事情都会有意义和目的。

所以，请记住，神在你生命中的建造过程不是要阻碍你，而是为了给你盼望和忍耐！

**祷告：主啊，无论我今天面对什么，求你帮助我记住，你掌管我的生命，万事发生都有你的美意。**
