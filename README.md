# /今日与恩典同行/

### 第106日 04/16 礼拜六

>你要逃避少年的私欲，同那清心铸告主的人追求公义、信德、仁爱、和平。 -提摩太后书2:22

20世纪90年代末，斯科特•奥格雷迪 (Scott O°Grady）因波斯尼亚上空的空战而名声大噪。几天来，这名被击落的美国飞行员在敌方境内躲过了抓捕。当直升机在奥格雷迪藏身地附近的空地上降落后，他并没有在灌木丛中放松下来。他甩掉疲惫，奋力穿过灌木丛，拔出武器，用尽全身力气奔向救援人员。

他的行动就像神要求我们在属灵争战中的表现一样。我们不可认为，因为神应许会赐下出路摆脱试探，我们在自己得拯救的过程中就无需付出努力。

当你身处敌方境内，要充分利用神恩赐的逃生手段，就是他提供的一切资源！

**祷告：主啊，我知道我生活在敌人的领地，面临许多试探。请你帮我充分利用你恩赐的逃生手段！**
