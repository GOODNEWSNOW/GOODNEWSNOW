# /今日与恩典同行/

### 第111日 04/21 礼拜四

>你们得不着，是因为你们不求。你们求也得不着，是因为你们妄求，要浪费在你们的宴乐中。 -雅各书4:26-3

信心的祷告取決于我们对信心和怀疑这两个词的理解程度。有时，我们会感到困惑，因为有好心人不断向我们举例，说有人祈祷某事发生——表现出坚定不移的信心，相信它必会发生——然后这事就发生了。

但是，当我们让神受制于我们人一时兴起的期许和欲望时，麻烦就来了。我们要重审这样一种要求神满足我们一切所愿的观念因为凭我们有限的智慧，我们所求的可能并非无限智慧的神想要给我们的！

所以，请记住，神总是会回应我们的祈铸。但如果你没有得到你所祈求的，请放心，因为神很可能在成就更好的事。

**祷告：天父，请教导我为你所希望的事情祈祷。求你帮助我相信你的回应远比我想要的好。**
