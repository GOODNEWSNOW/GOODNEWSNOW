# /今日与恩典同行/


### 第275 10/01 礼拜六

>因我们还软弱的时候，基督就按所定的日期为罪人死。为义人死，是少有的；为仁人死，或者有敢做的。惟有基督在我们还作罪人的时候为我们死，神的爱就在此向我们显明了。—罗马书5:6-8

大教堂和博物馆里到处都是宁静祥和的耶稣画像。许多人从描绘我们救主圣洁完美的艺术作品中得着极大安慰。但对于基督徒艺术家唐•蒂米埃 (Don Tiemier）来说，这些画像给他带来了信仰危机，因为他将神的完美与自己的罪性相比照。他陷入了深深的绝望，因他发现自己永远够不到如此完美的神。

可是后来他在罗马书5:6找到了平安，那里宣告了一个叫人得释放的真理：基督为罪人死。在这节经文中，蒂米埃与耶稣面对面，一点不觉得羞愧。通过这六个字，蒂米埃明白了耶稣愿意採救罪人。而这句话就是恩典的全部信息—借着耶稣基督在十字架上的牺牲，消除了绝望，迎来了平安。

**祷告：天父，因你的儿子在我还作罪人的时候为我而死，求你帮助我每一日感谢你的恩典，为他而活。**

### 第274 09/30 礼拜五

>但现今，你们既从罪里得了释放，作了神的奴仆，就有成圣的果子，那结局就是永生。因为罪的工价乃是死；惟有神的恩赐，在我们的主基督耶稣里，乃是永生。-罗马书6:22-23

一位家有两个青少年的母亲在教会青年教室摆放晚上的点心，她看了一眼白板。然后，她带着一丝愤怒，转身坚定地对带领青年的牧师说：“我的孩子不需要更多关于恩典的讲论。他们需要的是规矩。”

这位母亲的话听上去或许有些刺耳、武断甚至不客气，但带领青年的牧师没有感到烦扰。他知道她是一位敬虔的母亲，真的担心她的孩子——还有教会其他孩子。而且他知道其他父母也有这样的担心。

每当我们传讲，神无条件的爱给人出路得以逃离罪疚和罪的权势，就会产生这样的惧怕，即青少年和成人会滥用神的恩典。但我们必须相信，恩典胜过我们的罪和惧怕。

**祷告：主啊，感谢你赐下永恒的生命！求你帮助我今天用我在基督里的自由服侍你。**
- - -

### 第273 09/29 礼拜四 

>知道你们得赎，脱去你们祖宗所传流虚安的行为，不是凭着能坏的金银等物，乃是凭着基督的宝血，如同无瑕疵、无玷污的羔羊之血。——彼得前书1:18-19

当我刚开始和我妻子凯茜约会时，她是名专业的长笛演奏者。但我并不了解音乐家和她的乐器之间的关系。我还记得在一次音乐会后开车接她的情景。我打开后备箱，正要把长笛放进去——哦，看我都干了什么！那支笛子不能放在后备箱或后座上。凯茜的笛子在前排有一个荣誉的位置，就在我们之间。因为笛子是花了很高昂的价钱买的，所以她会特别小心地保管。

这略微反映了神是如何看待我们的。他因着自己的恩典，将他的爱定睛在我们身上，以极大的代价买赎我们——就是他自己儿子的性命。

**祷告：父啊，感谢你用耶稣基督的宝血买赎我们。求你帮助我以感恩和顺服来回应！**
- - -

### 第272 09/28 礼拜三 

>我小子们哪，我将这些话写给你们，是要叫你们不犯罪。若有人犯罪，在父那里我们有一位中保，就是那义者耶稣基督。他为我们的罪作了挽回祭，不是单为我们的罪，也是为普天下人的罪。——约翰一书2:1-2

我们都知道积非成是的謬误，那积是成非对吗？当然也不对。真正的悔改不是从做坏事转变为做好事。圣经清楚说明，真正的悔改是转离我们的罪，并进入耶稣的救赎恩典。借着神的恩典，我们所有的错误都被涂抹——得蒙赦免，不再被记念。

当我们意识到自己对罪的认识总是不准确、不完全的，我们对罪的懊悔也永远不够，那神的恩典涂抹我们一切的罪，就能给我们极大的安慰。

你可以放心，因为神的心大到足以遮盖你无法完全承认的事，他的恩典也宽广到足以除去你一切的怀疑和惧怕。

**祷告：耶稣，我承认我每日需要你的恩典。感谢你作我和父之间的中保，为我的罪作了挽回祭。**
- - -

### 第271 09/27 礼拜二 

>然而，人未曾信他，怎能求他呢？未曾听见他，怎能信他呢？没有传道的，怎能听见呢？——罗马书10:14

"导航员" (The Navigators)道森·特罗曼 (Dawson Trotman) 于1933年创立的一个国际事工。这一切都始于加州洛米塔 (Lomita)一个德士古加油站外发生的事。特罗曼开始向两个年轻水手传道，分享他的见证，并相信如果他们信靠神，神会使用他们。六个月后，这两个人说服特罗曼成为他们在西弗吉尼亚号战舰上的传教士。

目的地是？珍珠港。袭击发生前，有120到130人定期与特罗曼和那两名水手见面。但到了战争结束时，有800艘船都有“导航员”在带领查经。

特罗曼的故事深刻地提醒我们，神的工作始于小小的开端。他的恩典在背后以我们无法想象的方式运作！

**祷告：主啊，求你帮助我与我的家人、朋友、邻居分享好消息——并且支持其他国一家的宣教士。**
- - -

### 第270 09/26 礼拜一 

>神啊，我曾求告你，因为你必应允我；求你向我侧耳，听我的言语。求你显出你奇妙的慈爱来；你是那用右手拯救投靠你的脱离起来攻击他们的人。——诗篇17:6-7

最近，我有幸在大峡谷的科罗拉多河乘漂流筏顺流而下。正当季风季节，所以我们碰上了一场大暴雨。鸟云密布的时候，我比其他人更快拿到我的雨具。当雨点打在我们周围，我觉得自己就像地毯里的虫子一样舒服，直到雨水开始从我的背上流下来——只因有个漏水的缝。

我们都有漏水的缝。如果你不这么觉得，你就不会去导求帮助。当你承认自己有裂缝，向神呼求帮助，他的恩典就会开始流溢。他说：“不管你陷入怎样的困境，只要你说你需要我的帮助，我就在你身边。”

**祷告：亲爱的救主，我生命中有许多“裂缝”。当我向你呼求帮助，愿你的恩典更自由地在我生命流淌。**
