# /今日与恩典同行/

### 第171 06/20 礼拜一

>我们既有这许多的见证人，如同云彩围着我们，就当放下各样的重担，脱去容易缠累我们的罪，存心忍耐，奔那摆在我们前头的路程，仰望为我们信心创始成终的耶稣他因那摆在前面的喜乐，就轻看羞辱，忍受了十字架的苦难，便坐在神宝座的右边。——希伯来书12:1-2

当我看到儿子在越野跑比赛中体力下降的时候，就会大声鼓励他重振决心，专注目标。我知道，发出警告的声音或表现出我的沮丧，只会消磨他的力量——即使那可能会给他一时的激励。

同样，我们奔向神为我们设立的目标时，也要相信他的恩典不会因我们跑慢了或跌倒了就断绝。比起我来，神是位更称职的父亲，他充满恩典的鼓励在他儿女心中的回响也更加有力、满有慈爱且绵延不绝。

恩典不仅是神使我们完全的途径，我们也借此得激励、能以用丝毫不减的喜乐持续服侍他。

**祷告：耶稣，谢谢你在我奔走基督徒生命时每日鼓励我。愿你的鼓励给我力量和忍耐。**
