# /今日与恩典同行/

### 第161 06/10 礼拜五

>我儿，不要忘记我的法则；你心要谨守我的诚命；因为他必将长久的日子，生命的年数与平安，加给你。——箴言3:1-2

试想：我们若爱神，取悦他就会使我们自己喜悦。爱神与爱我们自己并不对立。当我们取悦我们最爱的那一位，我们就能寻见自己最大的喜悦。换句话说，满足我们最深爱的那一位，我们自己也能得着最深的满足。

所以，圣经勉励我们服侍神不要出于过于用力或勉强的无私，而要出于自我实现的喜乐和满足。当我们对神的爱成为心中最要紧的事，那么，用生命去荣耀他也就成了我们最大的喜乐！全心全意地爱神是最好地爱自己的终极表达！

**祷告：父啊，求你帮助我在今日所行的每一件事上都讨你喜悦，因为爱你我就能寻得最大的满足！**
