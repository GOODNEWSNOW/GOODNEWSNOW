# /今日与恩典同行/

### 第158 06/07 礼拜二

>你这论断人的，无论你是谁，也无可推诿。你在什么事上论断人，就在什么事上定自己的罪；因你这论断人的，自己所行却和别人一样。——罗马书2:1

在戏剧 《史上最佳圣诞汇演》中，主人公是赫德曼家的七个孩子。他们都是十足的不良少年——他们说谎、抽烟、骂人，还在学校欺负同学。他们从不去教堂。对小镇上的其他孩子来说，教堂是他们唯一可以享受些许平静的地方——直到赫德曼一家的孩子出现。

这几个孩子突然出现在主日学的课堂上，他们奇思怪想地问了老师好多问题，比如：“他们为什么叫他耶稣不叫他比尔？”“他们为什么不把圣诞节叫做伯利恒的复仇？”

当我们透过这些从未听闻恩典故事，却迫切需要恩典之人的眼睛来看圣诞节，就能看到我们自己也是多么需要恩典！

**祷告：父啊，求你帮助我不要去论断那些不认识你的人。我知道我并没有比他们好，我能得救完全是靠着你的恩典！**
