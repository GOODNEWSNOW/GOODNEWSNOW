# /今日与恩典同行/

### 第179 06/28 礼拜二

>又对他们说：“你们去吃肥美的，喝甘甜的，有不能预备的就分给他，因为今日是我们主的圣日。你们不要忧愁，因靠耶和华而得的喜乐是你们的力量。——尼希米记8:10

没什么比罪疚和羞愧这个双头怪兽更能拦阻我们与基督同行了！撒但深谙此道！你的控告者撒但最想看到你被这两种沉重的感觉所拖累和消耗。但罪疚和羞愧不是信心的动力。它们只会扼杀你的属灵能力，同时夺走你的盼望和喜乐。

圣经告诉我们“靠耶和华而得的喜乐是你们的力量”。所以，与其被拖累，不如让神的恩典推动你的基督徒之路。记住，你和神有正确的关系是耶稣的功劳！他在十字架上的工作为你铺平了道路。多少罪疚和羞愧都无法改变这一点！

**祷告：父啊，当撒但想要用罪疚和羞愧使我沮丧低落，求你帮助我喜乐，因为没有什么能使我与你在基督里的恩典隔绝！**
