# /今日与恩典同行/

### 第121日 05/01 主日

>你们作妻子的要顺服自己的丈夫；这样，若有不信从道理的丈夫，他们虽然不听道，也可以因妻子的品行被感化过来；这正是因看见你们有贞洁的品行和敬畏的心。——彼得前书3:1-2

三十多年来，希瑟一直忠心地为主和她的丈夫詹姆斯而活他是个刚硬的警长，不想与她的信仰有任何瓜葛。

我曾经问她，他丈夫这么无情地嫌弃她，时不时还嘲笑她的信仰，是什么让她一直这样舍己地为丈夫而活。她的回答很简单：当她成为基督徒后，她就爱上了耶稣。而这种爱使她想要她丈夫得着耶稣想要赐给他的。

当妻子顺服的时候，即使人的理性无法理解，而且似乎毫无价值，神的旨意也会帮助她。神知道他在做什么，今天，那个刚硬的警长詹姆斯在忠心服侍主。

让这个见证鼓励你像耶稣一样去爱吧！

**祷告：父啊，请使我能在配偶和其他人面前过敬虔的生活，让他们能看到你在我身上的改变，被吸引到耶稣基督跟前。**
