# /今日与恩典同行/

### 第109日 04/19 礼拜二

>我的弟兄们，你们落在百般试炼中，都要以为大喜乐；因为知道你们的信心经过试验，就生忍耐。但忍耐也当成功，使你们成全、完备，毫无缺欠。 -雅各书1:2-4

神要我们祷告的首要目标是转化我们的灵性，这一点有助于解释为什么神迟迟不回应一些祈求。我们可能希望环境改变，但神却希望我们改变！

我们想要快速解决问题，神却要我们成长！我们想要摆脱某个麻烦的同事，神却要我们增加对那人的爱。我们希望所有的问题和压力统统得到解决，但神希望我们学习在一切事上信靠他！

正如威廉•坦普尔（William Temple）写道：“神是完全的爱完全的智慧。我们祷告不是为了改变他的旨意，而是要使我们的意志合乎他的旨意。”

**祷告：主啊，帮助我按照你的旨意祷告，认识到你最大的愿望是我在灵性上成长，而不是改变环境。**
