# /今日与恩典同行/

### 第180 06/29 礼拜三

>耶和华是我的亮光，是我的探救，我还怕谁呢？耶和华是我性命的保障，我还惧谁呢？那作恶的就是我的仇敌，前来吃我肉的时候就绊跌仆倒。——诗篇27:1-2

让我们坦诚面对现实吧。我们都会面临危机。也许是大范围的危机：由政府官员的误导、自然灾害甚至恐怖分子所造成。也许是个人危机，比如健康问题、财物困难、事业挫败或棘手的人际关系。无论你今天面临什么样的处境，请记住，你并不孤单。

诗篇27篇提醒我们：“耶和华是我的亮光，是我的拯救，我还怕谁呢？耶和华是我性命的保障，我还惧谁呢？

所以，要将一切忧虑都卸给主，因为他顾念你。要相信神借着他恩典的供应，使万事互相效力，叫你得益处。

**祷告：天父，我知道，因为你与我同在，我永远不用惧怕。求你帮助我在大大小小的危机时刻都能信靠你的爱和恩典。**
