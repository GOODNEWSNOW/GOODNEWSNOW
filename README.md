# /今日与恩典同行/

### 第157 06/06 礼拜一

>因有一婴孩为我们而生；有一子赐给我们。政权必担在他的肩头上；他名称为“奇妙策士、全能的神、永在的父、和平的君”——以赛亚书9:6

神赐给他儿子许多称号：“全能的神““奇妙策士““和平的君“。耶稣的这些名称提醒我们，神为我们供应了我们无法为自己供应的。

神供应了我们需要的能力。他赐下我们想要的智谋。神差派和平的君来安慰我们恐惧的心，舒缓我们背负重担的灵，改变我们这个动荡混乱的世界。这就是福音。这就是神的恩典。这些都是他的供应——不仅是给那些自以为生命毫无问题的人，也给那些深知自己迫切需要的人！

**祷告：主啊，谢谢你在我软弱时赐我力量，在我困惑时赐我引导，在我焦虑不安时赐我基督里的平安！**
