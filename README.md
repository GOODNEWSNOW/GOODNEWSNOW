# /今日与恩典同行/

### 第110日 04/20 礼拜三

>你的话是我脚前的灯，是我路上的光。 -诗篇119:105

多年前，我必须做一个决定，是从事广播新闻工作还是上法学院。当我祷告时，我开始认识到我一直在以我能获得多少金钱和名声来考量我的选择。那些基督教原则我根本想都没想。

一位很有智慧的朋友建议我去读神学，好叫我回到正确的优先次序。而且，我承认，我去读的时候觉得这会影响我的事业。

但神是好的。他知道为什么要让我走上那条路。你看，就是在那里，我遇到了我的妻子，带领了一间历史悠久的教会，养育了敬虔的家庭，并在我曾经犹豫着要不要去的那所神学院成为了一名领袖。

所以，让神带你走他的路。我保证你不会后悔！

**祷告：父啊，当我展望未来时，愿你的话语和圣灵来引导我。**
