# /今日与恩典同行/

### 第135 05/15 主日

>我们有这宝贝放在瓦器里，要显明这莫大的能力是出于神，不是出于我们。我们四面受敌，却不被困住；心里作难，却不至失望；遭逼迫，却不被丢弃；打倒了，却不至死亡。身上常带着耶稣的死，使耶稣的生也显明在我们身上。——哥林多后书4:7-10

几年前，大学教授史蒂夫•科贝特 (Steve Corbet） 告诉我们，他从神那得了一个女儿，出生时就患有眼耳脊椎发育不良综合征，这种疾病会导致一些孩子出生时没有面部骨骼。科贝特夫妇的女儿出生时左侧没有面部骨骼，也没有左眼。史蒂夫说，他和家人在爱护和抚养他们的宝贝女儿初期，紧紧抓住了有两面的岩石——耶稣。

同样的挑战也摆在我们面前。当我们的试炼和失望过于我们所能受的，我们必须紧紧抓住耶稣的两面一面让我们看到神的伟大，他赐我们力量面对明天；另一面让我们看到神的良善，好叫我们可以信靠他面对今天。

**祷告：天父，我知道这个堕落的世界充满了伤痛和苦难，其中一些苦痛折磨着我和我的家人。求你帮助我相信你的良善，每天都依靠你的力量。**
_ _ _

### 第134 05/14 礼拜六

>因你们己经脱去旧人和旧人的行为，穿上了新人。这新人在知识上渐渐更新，正如造他主的形像。——歌罗西书3:96-10

我们在基督里是新造的人，神的灵住在我们里面，教导、训练并重新校正我们，叫我们长大成熟。这超自然的校正使我想起小时候玩的电动火车。

如果我想让火车反方向行驶，我不是用手去推，而是按一下线路上的开关，这样火车就会自动朝新方向继续行驶。

同样，圣灵也是以天然的努力无法达成的方式来改变你的心。当圣灵超自然地重新校正你的心去爱神、服侍神、顺服神，你会想要并且有能力去跟随。

**祷告：父啊，我旧人的生命已经过去，我在基督里有了新生命，为此我感谢你。愿这新生命激励我每天去爱你、服侍你、顺服你。**
