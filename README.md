# /今日与恩典同行/

### 第128 05/08 主日

>所以，你们若真与基督一同复活，就当求在上面的事；那里有基督坐在神的右边。你们要思念上面的事，不要思念地上的事。因为你们已经死了，你们的生命与基督一同藏在神里面。基督是我们的生命，他显现的时候，你们也要与他一同显现在荣耀里。——歌罗西书3:1-4

许多虚假属肉体的宗教都承诺奖励自私行为。但基督教却对信徒有不同的要求——为了我们在基督里丰盛的新生命而无私。身为信徒，我们用神的希望和梦想替代自己的希望和梦想。因为我们明白，我们最宝贵的财产是与基督同在的生命，这是借由圣灵得到的。

真正的信仰让我们看透世上幸福的幻象，引导我们领受无止尽的喜乐，就是那住在我们里面神奇妙的灵所带来的。而这一切都是神恩典的工作！

所以，今天就花点时间与喜乐和感恩共舞，因为知道无论你今生的境遇如何，你都因为神的恩典而永远安稳，蒙受无条件的爱。

**祷告：主耶稣，我永远感激你赐给我的丰盛生命和盼望。求你帮助我在今天充满喜乐和感恩！**
