# /今日与恩典同行/

### 第113日 04/23 礼拜六

>又当存敬畏基督的心，彼此顺服。 ——以弗所书5:21

我们每个人都拥有独特的个性、独特的才能和背景，神希望用这些来使我们的家、使我们彼此变得更好。如果你的配偶很难相处，你必须非常小心地面对所谓的错误——通常还要有极大的耐心。

要想理智又谦卑地面对错误，你必须相信神已经给了你的配偶恩赐，使你和你们的婚姻受益。

正因此故，圣经所说的彼此顺服不是压制恩赐，而是为了你们彼此的益处而充分使用恩赐。

你们若顺服彼此，就等于是让神使用你们的恩赐和才能，将他的良善和性情反映到你所爱之人的生命中去！

**祷告：主啊，我为我的配偶感谢你。在我们的婚姻中，帮助我用你所给的恩赐和才能，使我们彼此受益。**
