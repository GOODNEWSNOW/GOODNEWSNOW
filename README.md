# /今日与恩典同行/

### 第159 06/08 礼拜三

>愿颂赞归与我们的主耶稣基督的父神，就是发慈悲的父，赐各样安慰的神。我们在一切患难中，他就安慰我们，叫我们能用神所赐的安慰去安慰那遭各样患难的人。——哥林多后书1:3-4

每年日历都是从第1日开始。但日历翻过不代表我们的环境也跟着变了。也许你正为健康而担惊受怕，或者正遭遇财务危机。也许你正处在一段摇摆不定的关系中，或者工作上遇到了问题。

无论你现在面临什么，我希望你能在神荣耀的恩典中得安慰。因为他知道你故事的结局！我们身处的时代或许充满挑战，但是和许多前辈一样，我们必须紧紧抓住神的恩典和他能改变生命的真理。因为在那恩典里，我们的生命和环境都会被永远改变！

**祷告：天父，你知道我正在经历的困难。愿你的恩典每天赐给我安慰和力量。**
