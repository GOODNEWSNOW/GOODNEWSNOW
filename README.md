# /今日与恩典同行/

### 第132 05/12 礼拜四

>内中有一个见自己已经好了，就回来大声归荣耀与神，又俯伏在耶稣脚前感谢他；这人是撒马利亚人。耶稣说:"洁净了的不是十个人吗？那九个在哪里呢？除了这外族人，再没有别人回来归荣耀与神吗？"——路加福音17:15-18

几年前，一位年轻妈妈发现她儿子在姐姐房间里吃着泰迪熊巧克力，而那是别人送给她的礼物。小男孩像个走投无路的罪犯一样背靠着墙，他知道藏不住自己的罪行，也藏不住满脸的巧克力。于是，他开始抽泣，交代了自己做了什么。

姐姐回到家，这个小弟弟想要改正错误，就承认了自己所做的。姐姐把他搂在怀里，亲吻了他，还告诉他没事。此时，弟弟又哭又笑地用尽全力抱住他姐姐。

同样，我们承认自己所犯的罪时，就会因神恩慈的赦免而生出爱与感恩，进而用尽全力去拥抱主。

**祷告：主啊，我承认我有罪，该做的没做，不该做的却做了。谢谢你恩慈的爱与赦免洗净了我一切的罪！**
