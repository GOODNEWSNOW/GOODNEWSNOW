# /今日与恩典同行/

### 第125日 05/05 礼拜四

>所以，我们不丧胆。外体显然毀坏，内心却一天新似一天。我们这至暂至轻的苦楚，要为我们成就极重无比、永远的荣耀。——哥林多后书4:16-17

妥瑞氏综合症的破坏性影响对家庭来说很难承受，特别是要做我们一般人都认为理所当然的日常事情，比如外出吃饭。

就在一次这样的外出活动中，贾斯汀的母亲很快就把她患此病症的小儿子和他的姐妹们带出了餐厅。当一家人在躲避周围人的窃窃私语和目光时，贾斯汀的姐姐问道，他是否会一直这样。他们的母亲当时无言以答，后来伤心地向丈夫讲述了此事。

但贾斯汀的父亲慈爱地提醒妻子：“贾斯汀不会一直这样，因为荣耀会降临，贾斯汀会在基督里收获奖赏，而那奖赏会让今生的试炼成为遥远的记忆！”

**祷告：父啊，求你帮助我每天“不是顾念所见的，乃是顾念所不见的；因为所见的是暂时的，所不见的是永远的”（林后4:18)。**
