# /今日与恩典同行/

### 第130 05/10 礼拜二

>耶和华你的神是施行拯救、大有能力的主。他在你中间必因你欢欣喜乐，默然爱你，且因你喜乐而欢呼。——西番雅书3:17

一则老式汽车广告夸口说：“你开什么车决定了你是谁。”我不知道别人对这则广告的人物特征分析怎么看，我觉得这是对我的侮辱。也许我如此恼火是因为当时我在开一辆老式福特平托，它已经行驶超过16万公里了！它四个轮胎都得换，还要重新喷漆。

我们若认识到神供应了我们所需的一切，那么我们就明白了个重要启示：我们的价值不是基于我们目前的地位，而是基于我们在基督里的身份。

神说你在基督里是谁，你就是谁，因为对他来说，无比有价值——而不是你所拥有的。

**祷告：主啊，求你帮助我认识到，我的价值并不取决于我在世上的地位或别人对我的看法。谢谢你在基督里看我为无比有价值！**
