# /今日与恩典同行/

### 第170 06/19 主日

>不但如此，我也将万事当作有损的，因我以认识我主基督耶稣为至宝。我为他已经丢弃万事，看作粪士，为要得着基督；并且得以在他里面，不是有自己因律法而得的义，乃是有信基督的义，就是因信神而来的义。——腓立比书3:8-9

你知道吗？神是透过他儿子在十字架上所成就的来看待我们这些儿女。我们若相信耶稣为我们的罪受刑罚，我们就得与神和好。我们被称为义，也就是说，因着基督，神看我们好像我们从没有犯过罪！这不是因为我们做了什么，而是因为耶稣在十字架上所做的。当神看我们，他看到了他自己儿子的公义和良善。

我们相信这份恩典，神借这恩典选择看我们为披戴了基督的公义，这样的信心成了我们灵魂最大的盼望和无尽喜乐的源泉。

**祷告：主啊，我谢谢你使我披戴了你儿子耶稣基督的公义。愿这一事实成为盼望和无尽喜乐的源泉。**
