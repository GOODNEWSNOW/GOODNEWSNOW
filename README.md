# /今日与恩典同行/

### 第102日 04/12 礼拜二

>我今日所吩咐你的话都要记在心上，也要殷勤教训你的儿女。无论你坐在家里，行在路上，躺下，起来，都要谈论。 -申命记6:6-7

身为父母，我们都会犯错，但这并不意味着我们是坏父母。我和妻子凯茜也犯过不少错误。那些不当管教、不耐烦和判断失误的时候，我真希望神能从我们孩子的脑海中抹去。但就算他们记得，我也不会绝望。

圣经提醒你也提醒我，一时的错误不会毁掉我们孩子的生命。否则，我们可能什么也不敢去做，生怕毁掉他们。我们可以拥抱神的恩典，寻求宽恕然后再次尝试！

要感谢我们仁慈的天父也给我们第二次机会！请记住，天父无条件永恒的爱能消除我们的恐惧，不再害怕一个错误就会毁掉自己或孩子。

**祷告：父啊，我知道我也会犯错，可是感谢你一直在基督里原谅我。请帮助我对待自己的孩子时效法你无条件的爱。**
