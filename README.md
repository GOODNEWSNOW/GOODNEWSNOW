# /今日与恩典同行/

### 第154 06/03 礼拜五

>神爱世人，甚至将他的独生子赐给他们，叫一切信他的，不致灭亡，反得永生。因为神差他的儿子降世，不是要定世人的罪，乃是要叫世人因他得救。——约翰福音3:16-17

有时我们很难分清神的恩典与爱。爱关乎情感，它让你想要去安慰和关心他人，而恩典更关注供应。

恩典是礼物，是人不配得的神的恩惠。出于神对我们的喜爱，他供应他的恩典。因此，恩典与爱相互关联，但恩典的意义远不止于此！

恩典是神赐下他儿子，为我们的罪成为祭物，好叫我们能信靠他，在他里面有盼望——而不是在我们自己里面。

“神爱世人，甚至将他的独生子赐给他们，叫一切信他的，不致灭亡，反得永生。”

**祷告：父啊，谢谢你赐下你的儿子。求你帮助我今日和每一日都信靠基督、在基督里有盼望——而不是在我自己里面。**
